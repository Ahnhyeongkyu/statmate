t-test 결과 APA 7판으로 보고하는 법 (표 자동 생성 도구 포함)

[본문]:
==========================================
논문을 쓰다 보면 t-test 결과를 APA 7판 형식에 맞춰 보고하는 것이 꽤 까다롭습니다. 이탤릭체, 소수점 자릿수, 효과 크기까지 신경 써야 할 것이 한두 가지가 아닌데요.

오늘은 t-test 결과를 APA 7판에 맞게 보고하는 방법과, 이 과정을 자동으로 해주는 무료 도구를 소개하겠습니다.

■ APA 7판 t-test 보고 형식

독립표본 t-test의 경우 다음과 같이 보고합니다:
t(df) = t값, p = p값, d = Cohen's d

예시: t(58) = 2.45, p = .017, d = 0.63

보고 시 주의사항:
1. t와 p는 이탤릭체로 표기
2. p값이 .001 미만이면 p < .001로 표기
3. 효과 크기(Cohen's d)를 반드시 포함
4. 95% 신뢰구간 보고 권장

■ 결과표 작성 형식

APA 7판에서는 결과를 표로 정리할 때도 정해진 형식이 있습니다. 표 번호, 제목, 열 머리글, 각주 등 세세한 규칙을 지켜야 합니다.

이 표를 수동으로 만들면 보통 1~2시간이 소요되는데요.

■ 자동으로 APA 표 만드는 방법

StatMate(statmate.org)라는 무료 웹 도구를 사용하면 데이터를 입력하고 분석을 실행하면 APA 7판 형식의 결과표가 자동으로 생성됩니다.

주요 특징:
- 독립표본/대응표본 t-test 모두 지원
- Welch's t-test(등분산 가정 불필요) 기본 적용
- 효과 크기(Cohen's d) 자동 계산
- Shapiro-Wilk 정규성 검정 내장
- Word/PDF로 내보내기 가능
- 회원가입 불필요, 데이터는 브라우저에서만 처리

사용 방법:
1. statmate.org 접속
2. T-검정 선택
3. 데이터 입력 (직접 입력 또는 CSV 업로드)
4. 분석 실행 버튼 클릭
5. APA 형식 결과표 확인 → 복사 또는 Word 내보내기

논문 시즌에 시간을 절약할 수 있는 유용한 도구니 한번 사용해보세요!
