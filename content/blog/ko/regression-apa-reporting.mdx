## 회귀분석 결과 보고, 왜 정확해야 할까?

회귀분석은 양적 연구에서 가장 폭넓게 활용되는 통계 기법입니다. 공부 시간으로 시험 성적을 예측하는 단순한 분석부터, 여러 인구통계학적 변인이 직무 만족도에 미치는 영향을 동시에 모형화하는 복잡한 분석까지, 회귀분석은 사회과학 전반에서 핵심적인 역할을 합니다.

그런데 회귀분석 결과를 APA 형식에 맞게 보고하는 단계에서 많은 연구자가 어려움을 겪습니다. t-검정이나 ANOVA와 달리, 회귀분석에서는 전체 모형의 적합도, 개별 예측변인의 기여도, 비표준화 계수와 표준화 계수의 구분, 그리고 신뢰구간까지 다층적인 정보를 빠짐없이 제시해야 합니다. 이 중 하나라도 누락되면 학술지 심사에서 수정 요청을 받을 수 있습니다.

이 글에서는 APA 7판 기준으로 단순선형회귀와 다중회귀분석 결과를 보고하는 방법을 구체적인 숫자 예시와 함께 체계적으로 안내합니다.

## 단순선형회귀(Simple Linear Regression)

### 반드시 포함해야 할 요소

단순선형회귀 결과를 APA 형식으로 보고할 때 포함해야 하는 핵심 요소는 다음과 같습니다.

- **결정계수(*R*²)**: 예측변인이 결과변인 분산의 몇 퍼센트를 설명하는지를 나타내는 지표
- **F 통계량**: 전체 모형의 유의성 검정 결과, 이탤릭체로 *F*
- **자유도(df)**: 회귀 자유도와 잔차 자유도를 괄호 안에 표기
- **유의확률(*p*)**: 소수점 셋째 자리까지
- **비표준화 회귀계수(*B*)**: 기울기와 표준오차
- **표준화 회귀계수(*beta*)**: 표준화된 값

전체 모형의 기본 보고 형식은 아래와 같습니다.

> *R*² = .XX, *F*(df회귀, df잔차) = 통계량, *p* = 유의확률

### 연구 상황

대학생 120명을 대상으로 주간 공부 시간이 기말고사 점수를 예측하는지 검증하는 연구를 가정합니다.

### 전체 모형 보고

**올바른 보고 예시:**

> 주간 공부 시간이 기말고사 점수를 예측하는지 검증하기 위해 단순선형회귀분석을 실시하였다. 분석 결과, 전체 회귀모형은 통계적으로 유의하였다, *R*² = .34, *F*(1, 118) = 60.73, *p* < .001. 주간 공부 시간은 기말고사 점수 전체 분산의 34%를 설명하였다.

### 회귀계수 보고

> 주간 공부 시간은 기말고사 점수를 유의하게 예측하였다, *B* = 2.85, *SE* = 0.37, *t*(118) = 7.79, *p* < .001, 95% CI [2.12, 3.58]. 주간 공부 시간이 1시간 증가할 때마다 기말고사 점수가 평균 2.85점 상승하는 것으로 나타났다.

### 구성 요소 정리

| 요소 | 값 | 설명 |
|------|------|------|
| *R*² | .34 | 설명된 분산 비율 34%; 앞의 0 생략 |
| *F* | 60.73 | 전체 모형 F 통계량, 소수점 둘째 자리 |
| df | 1, 118 | 회귀 자유도(예측변인 수)와 잔차 자유도(*N* - *k* - 1) |
| *p* | < .001 | 매우 작은 값은 < .001로 표기 |
| *B* | 2.85 | 비표준화 기울기 |
| *SE* | 0.37 | 기울기의 표준오차 |
| *t* | 7.79 | 회귀계수의 t 통계량 |
| 95% CI | [2.12, 3.58] | *B*의 95% 신뢰구간 |

## 다중회귀분석(Multiple Regression)

### 예측변인이 여러 개일 때

다중회귀분석은 두 개 이상의 예측변인을 동시에 투입하여 결과변인을 예측합니다. APA 보고가 더 복잡해지는데, 전체 모형의 적합도뿐 아니라 각 예측변인의 고유한 기여도를 개별적으로 기술해야 하기 때문입니다.

### 연구 상황

학생 150명을 대상으로 공부 시간, 수업 출석률, 이전 학점(GPA)이 기말고사 점수를 예측하는지 검증하는 연구를 가정합니다.

### 전체 모형 보고

> 공부 시간, 수업 출석률, 이전 학점이 기말고사 점수를 예측하는지 검증하기 위해 다중회귀분석을 실시하였다. 전체 회귀모형은 통계적으로 유의하였으며, *R*² = .52, 수정된 *R*² = .51, *F*(3, 146) = 52.78, *p* < .001. 세 예측변인은 기말고사 점수 전체 분산의 52%를 설명하였다.

여기서 중요한 점은 다중회귀분석에서는 반드시 *R*²와 **수정된 *R*²(adjusted *R*²)**를 함께 보고해야 한다는 것입니다. 수정된 *R*²는 모형에 투입된 예측변인 수를 고려하여 보정한 값으로, 예측변인이 많을수록 *R*²가 인위적으로 커지는 문제를 방지합니다.

### 회귀계수 표(Coefficients Table)

개별 예측변인의 결과는 표로 정리하면 가독성이 높아집니다.

| 예측변인 | *B* | *SE* | *beta* | *t* | *p* | 95% CI |
|----------|------|------|--------|------|------|---------|
| (절편) | 12.45 | 5.32 | -- | 2.34 | .021 | [1.94, 22.96] |
| 공부 시간 | 1.92 | 0.31 | .38 | 6.19 | < .001 | [1.31, 2.53] |
| 출석률 | 0.28 | 0.08 | .22 | 3.50 | < .001 | [0.12, 0.44] |
| 이전 학점 | 8.74 | 1.85 | .29 | 4.72 | < .001 | [5.08, 12.40] |

### 개별 예측변인 해석

> 공부 시간은 기말고사 점수를 가장 강력하게 예측하는 변인이었다, *B* = 1.92, *SE* = 0.31, *beta* = .38, *t*(146) = 6.19, *p* < .001, 95% CI [1.31, 2.53]. 이전 학점 역시 기말고사 점수를 유의하게 예측하였다, *B* = 8.74, *SE* = 1.85, *beta* = .29, *t*(146) = 4.72, *p* < .001, 95% CI [5.08, 12.40]. 수업 출석률은 상대적으로 작지만 통계적으로 유의한 기여를 보였다, *B* = 0.28, *SE* = 0.08, *beta* = .22, *t*(146) = 3.50, *p* < .001, 95% CI [0.12, 0.44].

### 표준화 베타(*beta*)의 의미

표준화 회귀계수(*beta*)는 각 예측변인의 상대적 중요도를 비교할 때 사용합니다. *beta*는 원래 측정 단위가 아닌 표준편차 단위로 표현되므로, 척도가 다른 예측변인 간에도 직접 비교가 가능합니다.

위 예시에서 공부 시간의 *beta* = .38과 출석률의 *beta* = .22를 비교하면, 다른 변인을 통제한 상태에서 공부 시간이 기말고사 점수에 더 큰 고유한 기여를 하고 있음을 알 수 있습니다.

## 유의하지 않은 결과 보고

회귀분석이 유의하지 않은 경우에도 동일한 형식으로 전체 모형 통계량과 개별 회귀계수를 빠짐없이 보고합니다.

> 수면 시간이 기말고사 점수를 예측하는지 검증하기 위해 단순선형회귀분석을 실시하였다. 분석 결과, 수면 시간은 기말고사 점수를 유의하게 예측하지 못하였다, *R*² = .02, *F*(1, 118) = 2.41, *p* = .123. 회귀계수 역시 유의하지 않았다, *B* = 0.95, *SE* = 0.61, *t*(118) = 1.55, *p* = .123, 95% CI [-0.26, 2.16].

## 회귀분석 APA 보고에서 흔한 실수

### 1. 다중회귀에서 수정된 R² 누락

단순회귀에서는 *R*²만 보고해도 충분하지만, 다중회귀에서는 **수정된 *R*²**를 반드시 함께 보고해야 합니다. 수정된 *R*²는 예측변인 수를 고려하여 보정된 값이므로, 모형의 실질적 설명력을 더 정확하게 반영합니다. 이를 누락하는 것은 매우 흔한 실수입니다.

### 2. B와 beta 혼동

*B*(비표준화 계수)와 *beta*(표준화 계수)는 서로 다른 정보를 전달합니다. *B*는 예측변인의 원래 단위에서 결과변인이 얼마나 변하는지를 나타내고, *beta*는 표준편차 단위로 변환된 값이므로 예측변인 간 상대적 비교에 적합합니다. 어떤 계수를 보고하는지 항상 명확히 표기하고, 가능하면 두 가지 모두 포함하세요.

### 3. *p* = .000으로 보고

SPSS나 R 등 통계 소프트웨어에서 *p* = .000으로 출력되는 경우가 있습니다. 이는 *p*가 정확히 0이라는 의미가 아니므로, 반드시 ***p* < .001**로 표기해야 합니다.

### 4. 신뢰구간 누락

APA 7판에서는 회귀계수의 95% 신뢰구간을 보고하도록 강력히 권장합니다. 신뢰구간은 추정치의 정밀도에 대한 정보를 제공하며, *p* 값만으로는 전달할 수 없는 중요한 맥락을 더해 줍니다.

### 5. 절편(Intercept) 누락

절편은 대개 해석의 초점이 아니지만, 회귀계수 표에는 반드시 포함해야 합니다. 심사자들은 절편이 포함된 완전한 계수표를 기대합니다.

### 6. 가정 검토 미보고

회귀분석 결과를 보고하기 전에 핵심 가정을 확인해야 합니다. 선형성, 잔차의 독립성, 등분산성(잔차 분산의 일정성), 잔차의 정규성, 그리고 다중공선성의 부재(다중회귀의 경우)를 확인하세요. 가정 위반이 있었다면 결과 섹션에서 이를 언급하고 어떤 교정 조치를 취했는지 기술하는 것이 바람직합니다.

## APA 형식 회귀분석 보고 체크리스트

논문 제출 전에 다음 항목을 점검하세요.

- 전체 모형의 *R*²를 보고했는가 (다중회귀의 경우 수정된 *R*²도 포함)
- *F* 통계량과 올바른 자유도를 포함했는가
- 전체 모형의 정확한 *p* 값을 보고했는가
- 회귀계수 표에 *B*, *SE*, *beta*, *t*, *p*, 95% CI를 모두 포함했는가
- 절편(Intercept) 행이 계수표에 포함되어 있는가
- 비표준화 계수(*B*)와 표준화 계수(*beta*)를 명확히 구분하여 표기했는가
- 모든 통계 기호(*R*², *F*, *B*, *beta*, *t*, *p*)를 이탤릭체로 표기했는가
- 회귀분석의 기본 가정 검토 결과를 본문에 언급했는가

## 다중회귀분석을 APA 형식으로 보고하기

### 단순회귀와 다중회귀의 보고 차이

단순회귀에서는 예측변인이 하나이므로 전체 모형 검정과 회귀계수 검정이 동일한 결과를 산출합니다. 반면 다중회귀에서는 두 가지 층위의 보고가 필요합니다. 첫째, 전체 모형의 *F* 검정은 예측변인 집합이 전체적으로 유의한 분산을 설명하는지를 나타냅니다. 둘째, 각 예측변인은 다른 예측변인을 통제한 후의 고유한 기여도를 보여주는 개별 *t* 검정, 표준오차, 신뢰구간을 가집니다.

예측변인들이 분산을 공유하기 때문에 개별 *R*² 값의 합은 모형 전체의 *R*²를 초과하는 것이 일반적입니다. 이것이 다중회귀에서 수정된 *R*²를 보고해야 하는 이유이며, 개별 예측변인을 해석할 때 고유 기여와 공유 기여를 구분하는 표현이 필요한 이유입니다.

### 예측변인 2개의 APA 보고 예시

주간 운동 시간과 수면 질이 직장인 100명의 스트레스 수준을 예측하는지 검증하는 연구를 가정합니다.

> 주간 운동 시간과 수면 질이 지각된 스트레스 수준을 예측하는지 검증하기 위해 다중회귀분석을 실시하였다. 전체 회귀모형은 통계적으로 유의하였다, *R*² = .28, 수정된 *R*² = .27, *F*(2, 97) = 18.45, *p* < .001. 두 예측변인은 지각된 스트레스 분산의 28%를 설명하였다.

### 개별 예측변인 보고

각 예측변인에 대해 비표준화 계수 *B*, 표준오차, 표준화 계수 *beta*, 잔차 자유도가 포함된 *t* 통계량, *p* 값, 95% 신뢰구간을 보고합니다.

> 주간 운동 시간은 스트레스 수준을 유의하게 예측하였다, *B* = -1.47, *SE* = 0.38, *beta* = -.34, *t*(97) = -3.87, *p* < .001, 95% CI [-2.22, -0.72]. 수면 질을 통제한 상태에서 주간 운동 시간이 1시간 증가할 때마다 지각된 스트레스가 1.47점 감소하였다.

> 수면 질 역시 스트레스 수준을 유의하게 예측하였다, *B* = -2.83, *SE* = 0.71, *beta* = -.35, *t*(97) = -3.99, *p* < .001, 95% CI [-4.24, -1.42]. 운동 시간을 통제한 상태에서 수면 질이 1단위 향상될 때 지각된 스트레스가 2.83점 감소하였다.

### R² 변화량(Delta R²) 해석

위계적 회귀분석에서 예측변인을 단계적으로 투입할 때, *R*² 변화량(*Delta R*²)을 보고하여 각 단계에서 추가로 설명되는 분산 비율을 제시합니다.

> 1단계에서 주간 운동 시간을 투입하였으며 스트레스 분산의 15%를 설명하였다, *R*² = .15, *F*(1, 98) = 17.29, *p* < .001. 2단계에서 수면 질을 추가 투입하였으며, 모형은 스트레스 분산의 13%를 추가로 설명하였다, *Delta R*² = .13, *F*-change(1, 97) = 19.12, *p* < .001.

이 접근법은 통제변인을 먼저 투입하고 주요 예측변인을 나중에 투입하는 등 이론적 근거에 따라 변인 투입 순서를 정할 때 유용합니다.

## 회귀분석 가정 검증을 APA로 보고하기

### 5가지 핵심 가정

회귀분석 결과를 제시하기 전에 다음 가정들을 검증하고, 결과 섹션에서 간략히 언급해야 합니다.

**1. 선형성.** 각 예측변인과 결과변인 간의 관계가 선형이어야 합니다. 산점도와 부분회귀 도표를 검토하세요. 비선형성이 관찰되면 변수 변환이나 다항식 항의 추가를 고려합니다.

**2. 잔차의 독립성.** 잔차들이 서로 상관되어서는 안 됩니다. 이 가정은 종단 데이터나 군집 데이터에서 특히 중요합니다. Durbin-Watson 통계량을 보고합니다:

> Durbin-Watson 통계량은 1.92로, 잔차 간 실질적 자기상관이 없음을 나타냈다(2.0에 가까운 값은 독립적 오차를 시사).

**3. 등분산성(Homoscedasticity).** 잔차의 분산이 모든 예측값에 걸쳐 일정해야 합니다. 잔차 대 예측값 산점도를 검토하세요. 깔때기 형태가 관찰되면 이분산성이 존재할 가능성이 있습니다.

> 표준화 잔차와 표준화 예측값의 산점도가 무작위 패턴을 보여, 등분산성 가정이 충족된 것으로 판단하였다.

**4. 잔차의 정규성.** 잔차가 대략적으로 정규분포를 따라야 합니다. 잔차의 히스토그램이나 Q-Q 도표를 검토하고, 필요하면 Shapiro-Wilk 검정을 실시합니다.

> 표준화 잔차의 히스토그램과 Q-Q 도표를 검토한 결과, 대략적으로 정규분포를 따르는 것으로 확인되었다.

**5. 다중공선성(Multiple Regression 한정).** 예측변인들이 서로 지나치게 높은 상관을 보여서는 안 됩니다. 각 예측변인의 분산팽창인자(VIF)를 보고합니다:

> 모든 예측변인의 VIF 값은 1.12에서 2.34 사이로, 일반적으로 사용되는 임계값 10보다 훨씬 낮아 다중공선성 문제가 없음을 확인하였다.

### 가정이 위반되었을 때 보고하는 방법

가정이 위반된 경우, 위반 내용과 교정 조치를 기술합니다. 주요 전략은 다음과 같습니다.

- **비선형성**: 로그 또는 제곱근 변환 적용, 다항식 항 추가
- **자기상관** (Durbin-Watson이 2.0에서 크게 벗어남): 일반화 최소제곱법(GLS) 사용 또는 시차 변수 추가
- **이분산성**: 이분산-일치 표준오차(HC3 등) 또는 가중최소제곱법(WLS) 사용
- **잔차의 비정규성**: 변수 변환 또는 부트스트랩 신뢰구간 사용
- **다중공선성** (VIF 10 이상): 상관이 높은 변인 제거 또는 결합, 능형회귀(Ridge Regression) 사용

위반을 보고하는 예시:

> 잔차 도표의 시각적 검토 결과, 이분산성이 존재할 가능성이 확인되었다. 이를 보정하기 위해 모든 계수 검정에 이분산-일치 표준오차(HC3)를 적용하였다. 강건 표준오차를 적용한 후에도 유의한 예측변인의 패턴은 동일하게 유지되었다.

## 회귀계수의 신뢰구간

### APA 7판이 신뢰구간을 권장하는 이유

APA 7판은 신뢰구간을 강조하는데, 이는 신뢰구간이 점추정값과 추정의 정밀도라는 두 가지 정보를 동시에 전달하기 때문입니다. 좁은 CI는 정밀한 추정을 나타내고, 넓은 CI는 불확실성을 시사합니다. 또한 CI는 회귀계수가 0과 통계적으로 유의하게 다른지 직접 보여줍니다: 95% CI가 0을 포함하지 않으면 해당 예측변인은 .05 수준에서 유의합니다.

### 신뢰구간이 포함된 보고 예시

> 주간 공부 시간은 기말고사 점수를 유의하게 예측하였다, *B* = 2.34, 95% CI [0.89, 3.79], *t*(118) = 3.21, *p* = .002. 신뢰구간은 모집단에서의 실제 기울기가 추가 공부 1시간당 0.89점에서 3.79점 사이에 위치할 것으로 추정됨을 나타낸다.

### 신뢰구간 해석 방법

- **CI가 0을 포함하지 않음**: 해당 알파 수준에서 회귀계수가 통계적으로 유의함
- **CI가 0을 포함함**: 회귀계수가 통계적으로 유의하지 않음
- **좁은 CI**: 추정이 정밀함 (대표본이나 낮은 변동성에 기인하는 경우가 많음)
- **넓은 CI**: 추정이 부정확함 (더 큰 표본이나 향상된 측정이 필요)

### 다중 예측변인에 대한 완전한 APA 예시

> 공부 시간은 시험 점수를 유의하게 예측하였다, *B* = 1.92, 95% CI [1.31, 2.53], *beta* = .38, *t*(146) = 6.19, *p* < .001. 이전 학점 역시 유의한 예측변인이었다, *B* = 8.74, 95% CI [5.08, 12.40], *beta* = .29, *t*(146) = 4.72, *p* < .001. 출석률은 상대적으로 작지만 유의한 기여를 보였다, *B* = 0.28, 95% CI [0.12, 0.44], *beta* = .22, *t*(146) = 3.50, *p* < .001.

세 예측변인 모두의 신뢰구간이 0을 포함하지 않으며, 이는 세 변인 모두 통계적으로 유의한 것과 일치합니다.

## 표준화 계수 vs. 비표준화 계수

### B와 Beta, 언제 보고할까?

비표준화 계수(*B*)와 표준화 계수(*beta*)는 각각 다른 목적을 가지고 있으며, APA는 가능하면 둘 다 보고할 것을 권장합니다.

### 비표준화 계수(B)

*B*는 예측변인이 원래 단위에서 1단위 변화할 때 결과변인의 예측된 변화량을 나타냅니다. 원래 측정 단위가 유지되므로 실질적 해석에 적합합니다.

> 주간 공부 시간이 1시간 증가할 때마다 기말고사 점수가 2.85점 상승하였다(*B* = 2.85).

독자가 시간과 시험 점수라는 의미 있는 단위로 효과의 크기를 즉시 이해할 수 있습니다. 이는 년, 원, kg 등 직관적인 측정 단위를 사용하는 경우에 특히 유용합니다.

### 표준화 계수(Beta)

*Beta*는 예측변인이 1표준편차 변화할 때 결과변인이 표준편차 단위로 얼마나 변하는지를 나타냅니다. 원래 척도가 제거되므로 모형 내에서 예측변인의 상대적 중요도를 직접 비교할 수 있습니다.

> 공부 시간의 상대적 기여(*beta* = .38)가 출석률(*beta* = .22)보다 시험 점수 예측에 더 강한 것으로 나타났다.

서로 다른 척도로 측정된 예측변인의 상대적 강도를 비교할 때 *beta*를 사용합니다. 예를 들어, 시간(0~40)과 출석률(0~100%)을 비교하는 것은 표준화 계수를 통해서만 의미 있습니다.

### APA 형식으로 두 계수 모두 보고하기

> 공부 시간은 시험 점수를 유의하게 예측하였다, *B* = 1.92, *SE* = 0.31, *beta* = .38, *t*(146) = 6.19, *p* < .001, 95% CI [1.31, 2.53].

이 하나의 문장에 실질적 해석을 위한 *B*와 상대적 비교를 위한 *beta*가 모두 포함되어 있으며, 표준오차, 검정 통계량, 유의확률, 신뢰구간까지 제시되어 있습니다.

## 회귀분석 결과의 APA 표 형식

### 표와 본문 서술의 구분

예측변인이 하나인 단순회귀에서는 본문 서술로 충분합니다. 예측변인이 2개 이상이면 표를 사용하세요. 3개 이상의 예측변인이 있을 때는 본문만으로 보고하면 가독성이 크게 떨어지므로 표가 필수적입니다.

### 완전한 APA 회귀계수 표

잘 구성된 APA 회귀 결과표의 열 구성은 다음과 같습니다.

| 변인 | *B* | *SE* | *beta* | *t* | *p* | 95% CI |
|------|------|------|--------|------|------|---------|
| (절편) | 12.45 | 5.32 | -- | 2.34 | .021 | [1.94, 22.96] |
| 공부 시간 | 1.92 | 0.31 | .38 | 6.19 | < .001 | [1.31, 2.53] |
| 출석률 | 0.28 | 0.08 | .22 | 3.50 | < .001 | [0.12, 0.44] |
| 이전 학점 | 8.74 | 1.85 | .29 | 4.72 | < .001 | [5.08, 12.40] |

표 아래에 모형 요약 주석을 추가합니다:

> *주.* *R*² = .52, 수정된 *R*² = .51, *F*(3, 146) = 52.78, *p* < .001.

### 표 작성 지침

- APA 표 형식을 사용하여 머리글 행 위, 아래와 표 하단에 수평선을 배치합니다
- APA 표에서는 세로선을 사용하지 않습니다
- 열 머리글의 모든 통계 기호를 이탤릭체로 표기합니다: *B*, *SE*, *beta*, *t*, *p*
- 절편을 첫 행에 배치하고, 그 아래에 예측변인을 이론적 중요도 또는 투입 순서대로 나열합니다
- *B*와 *SE*는 소수점 둘째 자리까지, *beta*는 앞의 0 없이 소수점 둘째 자리까지 보고합니다
- 표 번호와 제목을 포함합니다 (예: "표 1. 기말고사 점수를 예측하는 다중회귀분석 결과")

### 모형 요약 행

일부 학술지에서는 계수표 위 또는 아래에 별도의 모형 요약 섹션을 선호합니다. 포함해야 할 항목은 다음과 같습니다.

- *R*²와 수정된 *R*²
- 자유도가 포함된 전체 *F* 통계량
- 모형의 *p* 값
- 추정의 표준오차(선택 사항이지만 정보가치가 높음)

## 자주 묻는 질문(FAQ)

### R²와 수정된 R²의 차이는 무엇인가요?

*R*²는 모든 예측변인이 결합하여 설명하는 분산의 비율을 보여줍니다. 수정된 *R*²는 예측변인의 수를 고려하여 보정하며, 불필요한 변인이 포함된 모형에 패널티를 부여합니다. 예측변인 수가 다른 모형을 비교할 때는 수정된 *R*²를 사용하세요. 예측변인이 하나인 단순회귀에서는 두 값이 거의 동일하지만, 다중회귀에서 수정된 *R*²는 항상 *R*²와 같거나 작으며, 의미 없는 예측변인이 많을수록 그 차이가 커집니다.

### 음수 회귀계수는 어떻게 해석하나요?

음수 계수는 예측변인이 1단위 증가할 때 다른 예측변인을 통제한 상태에서 결과변인이 해당 계수 값만큼 감소한다는 의미입니다. 부호는 관계의 방향을 나타내는 것이지 강도를 나타내는 것이 아닙니다. 예를 들어, 스트레스를 예측하는 운동 시간의 *B* = -1.47은 운동 1시간 증가가 스트레스 1.47점 감소와 관련된다는 의미입니다.

### 표준화 계수와 비표준화 계수 중 어느 것을 보고해야 하나요?

APA는 둘 다 보고할 것을 권장합니다. 비표준화 계수(*B*)는 원래 측정 단위를 유지하여 실질적 해석에 적합합니다. 예를 들어, 공부 1시간 추가가 시험 2.85점 상승을 예측한다는 것을 독자가 이해할 수 있습니다. 표준화 계수(*beta*)는 모형 내에서 예측변인의 상대적 중요도를 비교할 수 있게 하며, 이는 예측변인의 척도가 서로 다를 때 필수적입니다.

### VIF가 10을 초과하면 어떤 의미인가요?

분산팽창인자(VIF)가 10을 초과하면 심각한 다중공선성이 존재함을 나타내며, 이는 둘 이상의 예측변인이 매우 높은 상관을 가진다는 의미입니다. 이로 인해 표준오차가 부풀려져 개별 계수 검정의 신뢰성이 떨어지고, 계수의 부호가 뒤바뀌거나 유의한 효과가 가려질 수 있습니다. 상관이 높은 예측변인 중 하나를 제거하거나, 합성 변인으로 결합하거나, 능형회귀를 사용하는 것을 고려하세요.

### 범주형 예측변인으로 회귀분석을 할 수 있나요?

가능합니다. 범주형 변인을 더미변수(0/1 코딩)로 변환하면 됩니다. *k*개 범주를 가진 변인은 *k* - 1개의 더미변수가 필요합니다. 모든 더미변수가 0인 경우가 참조 범주(reference category)가 됩니다. 예를 들어, 3수준 교육 변인(고졸, 학사, 대학원)은 2개의 더미변수가 필요하며, 하나의 수준이 참조 집단으로 기능합니다.

### 모형이 유의하지 않을 때 어떻게 보고하나요?

유의한 모형과 동일한 통계량을 보고합니다: *F* 통계량, 자유도, *p* 값, *R*². 결과가 유의하지 않더라도 전체 모형과 개별 계수를 빠짐없이 보고해야 합니다.

> 회귀모형은 통계적으로 유의하지 않았다, *F*(2, 97) = 1.45, *p* = .240, *R*² = .03. 주간 운동 시간(*B* = -0.52, *p* = .187)과 수면 질(*B* = -0.84, *p* = .291) 모두 스트레스 수준을 유의하게 예측하지 못하였다.

### 단순회귀와 다중회귀의 차이는 무엇인가요?

단순회귀는 하나의 예측변인으로 결과변인을 설명하고, 다중회귀는 두 개 이상의 예측변인을 동시에 사용합니다. 다중회귀는 각 예측변인이 설명하는 공유 분산과 고유 분산을 고려하므로, 다중회귀에서의 개별 계수는 각각의 단순회귀에서 얻은 계수와 다를 수 있습니다. 또한 다중회귀는 수정된 *R*²를 제공하며, 이는 단순회귀에서는 불필요합니다.

### 회귀계수의 신뢰구간을 보고해야 하나요?

그렇습니다. APA 7판은 회귀계수의 95% 신뢰구간을 보고할 것을 권장합니다. 신뢰구간은 추정의 정밀도와 계수가 0과 유의하게 다른지에 대한 정보를 제공합니다. 95% CI가 0을 포함하지 않으면 해당 계수는 .05 수준에서 통계적으로 유의합니다. 또한 신뢰구간은 독자가 효과의 타당한 범위를 파악하여 실질적 유의성을 평가하는 데 도움을 줍니다.
