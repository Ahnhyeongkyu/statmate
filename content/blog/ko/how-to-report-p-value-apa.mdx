## APA에서 p값 보고가 중요한 이유

모든 양적 연구 논문에 p값이 포함되지만, 이는 학술 글쓰기에서 가장 자주 잘못 보고되는 통계량 중 하나입니다. 학술지는 부정확한 서식 때문에 원고를 반려합니다. 심사자는 *p* = .000으로 쓰거나 소수점 앞에 0을 추가하는 논문에 지적을 합니다. 독자는 통계적 유의성과 실질적 중요성을 혼동하는 저자에 대한 신뢰를 잃습니다.

미국심리학회는 *Publication Manual of the American Psychological Association, 7th Edition*(2020)에서 정확한 규칙을 수립하여 이러한 문제를 다루었습니다. 이 규칙들은 임의적인 스타일 선호가 아닙니다. 행동과학, 사회과학, 보건과학 전반에서 통계적 증거의 명확성, 일관성, 정확한 커뮤니케이션을 보장하기 위해 존재합니다.

p값 보고를 올바르게 하는 것이 중요한 세 가지 이유:

1. **재현 가능성.** 정확한 p값을 보고하면 다른 연구자들이 유의/비유의의 이분법적 라벨에 의존하지 않고 스스로 증거의 강도를 평가할 수 있습니다.
2. **메타분석.** 여러 연구의 결과를 통합하는 연구자에게는 정확한 p값과 효과크기가 필요합니다. *p* < .05만 보고하면 향후 메타분석가가 필요로 하는 정보가 파괴됩니다.
3. **출판 기준.** 심리학, 교육학, 간호학, 공중보건학의 주요 학술지가 APA 서식을 시행합니다. 부정확한 p값 보고는 학술적 관례에 대한 미숙함을 신호하며 출판을 지연시키거나 저해할 수 있습니다.

이 가이드는 APA 7판 형식으로 p값을 보고하는 데 필요한 모든 규칙, 모든 일반적 검정, 모든 실수를 다룹니다.

## p값의 핵심 APA 규칙

APA Publication Manual(Section 6.36)은 p값을 제시하는 방법을 정확히 규정합니다. 다섯 가지 규칙이 있으며, 모두 타협의 여지가 없습니다.

### 규칙 1: 소수점 이하 두세 자리까지 정확한 p값 보고

APA 7판은 소수점 이하 두세 자리로 반올림한 정확한 p값을 요구합니다. 세 자리가 가장 일반적인 관례이며 가장 많은 정보를 제공합니다.

- 올바름: *p* = .032
- 올바름: *p* = .007
- 부적절: *p* = .03(부정확)
- 부적절: *p* < .05(정확하지 않음)

정확한 값을 보고하면 독자가 스스로 증거를 평가할 수 있습니다. *p* = .048과 *p* = .002는 모두 ".05 수준에서 유의"하지만 매우 다른 이야기를 합니다.

### 규칙 2: .001 미만이면 *p* < .001 사용

p값이 .001보다 작을 때는 정확한 값을 보고하지 않습니다. 대신 *p* < .001로 씁니다.

- 올바름: *p* < .001
- 부적절: *p* = .000(확률은 절대 정확히 0이 아님)
- 부적절: *p* = .0001(소수점 이하 자릿수 초과)
- 부적절: *p* = 0.000(두 가지 오류: 소수점 앞 0과 불가능한 값)

SPSS 등의 통계 소프트웨어는 소수점 세 자리에서 반올림하면 0이 되는 경우 "Sig. = .000"을 표시합니다. 이는 반올림 인공물이며 실제 확률이 0인 것이 아닙니다. 실제 p값은 .00004나 .0000001일 수 있지만, APA 형식은 단순히 *p* < .001을 요구합니다.

### 규칙 3: 소수점 앞의 0 생략

p값은 0과 1 사이에 범위가 있으며 1.0에 도달하거나 초과할 수 없으므로, APA 양식에서는 소수점 앞의 0을 생략합니다.

- 올바름: *p* = .042
- 부적절: *p* = 0.042

이 규칙은 절대값이 1.0을 초과할 수 없는 모든 통계량에 적용됩니다. 여기에는 p값, 상관계수(*r*), 비율, Cronbach's alpha가 포함됩니다. Cohen's *d*, 평균, 표준편차처럼 1.0을 초과할 수 있는 통계량에는 적용되지 않습니다.

### 규칙 4: *p* 이탤릭체

APA 형식에서 모집단 모수에서 유래하거나 관례적 약어로 사용되는 통계 기호는 이탤릭체로 표기합니다. p값의 *p*는 항상 이탤릭체입니다.

- 올바름: *p* = .017
- 부적절: p = .017(이탤릭체 아님)

이탤릭체로 표기해야 하는 다른 통계 기호로는 *t*, *F*, *r*, *R*², *d*, *M*, *SD*, *N*, *n*, *df*, 그리고 편 에타제곱(*η*p²) 등의 통계량으로 사용되는 그리스 문자가 있습니다.

### 규칙 5: 연산자 앞뒤에 공백

등호 또는 부등호 앞뒤에 공백을 둡니다.

- 올바름: *p* = .032
- 올바름: *p* < .001
- 부적절: *p*=.032(공백 없음)
- 부적절: *p*<.001(공백 없음)

이는 =, <, >, 뺄셈 기호를 포함한 APA 양식의 모든 수학 연산자에 적용됩니다.

### 요약: 올바른 vs. 부적절한 p값 서식

| 올바름 | 부적절 | 부적절한 이유 |
|--------|--------|--------------|
| *p* = .032 | p = .032 | *p*는 이탤릭체여야 함 |
| *p* = .032 | *p* = 0.032 | 1을 초과할 수 없는 값은 소수점 앞 0 생략 |
| *p* = .032 | *p* = .03 | 소수점 이하 세 자리까지 반올림 |
| *p* < .001 | *p* = .000 | 확률은 절대 정확히 0이 아님 |
| *p* < .001 | *p* = .0001 | 매우 작은 값은 *p* < .001 사용 |
| *p* = .032 | *p*=.032 | = 앞뒤에 공백 필요 |
| *p* = .871 | *p* = n.s. | 비유의해도 정확한 값 보고 |

## 주요 검정별 p값 보고

각 통계 검정에는 고유한 APA 보고 템플릿이 있습니다. 아래는 가장 흔한 7가지 검정에 대한 구체적인 수치 예시입니다. 모든 예시는 APA 7판이 요구하는 검정 통계량, 자유도, 정확한 p값, 효과크기를 포함합니다.

### 독립표본 t-검정

**템플릿:** *t*(df) = X.XX, *p* = .XXX, *d* = X.XX

**예시(유의함):**

> 중재 집단 학생(*M* = 82.4, *SD* = 11.3)이 통제 집단(*M* = 74.1, *SD* = 12.7)보다 유의하게 높은 점수를 보였다, *t*(78) = 3.12, *p* = .003, *d* = 0.70.

**예시(비유의):**

> 카페인 집단(*M* = 312 ms, *SD* = 45)과 위약 집단(*M* = 319 ms, *SD* = 51) 간에 통계적으로 유의한 반응시간 차이가 없었다, *t*(62) = 0.61, *p* = .544, *d* = 0.15.

Cohen's *d*는 1.0을 초과할 수 있으므로 소수점 앞 0을 사용합니다.

### 대응표본 t-검정

**템플릿:** *t*(df) = X.XX, *p* = .XXX, *d* = X.XX

**예시:**

> 불안 점수가 사전검사(*M* = 38.2, *SD* = 8.4)에서 사후검사(*M* = 31.7, *SD* = 7.9)로 유의하게 감소하였다, *t*(44) = 4.58, *p* < .001, *d* = 0.68.

대응표본 t-검정의 자유도는 *n* - 1이며, *n*은 대응 쌍의 수입니다.

### 일원분산분석(One-Way ANOVA)

**템플릿:** *F*(df1, df2) = X.XX, *p* = .XXX, *η*p² = .XX

**예시:**

> 세 부서 간 직무 만족도에 통계적으로 유의한 차이가 있었다, *F*(2, 87) = 5.94, *p* = .004, *η*p² = .12.

첫 번째 자유도(2)는 집단 수 - 1입니다. 두 번째(87)는 전체 표본 크기 - 집단 수입니다. 옴니버스 F가 유의하면 사후 비교를 실시합니다.

### 카이제곱 독립성 검정

**템플릿:** *χ*²(df, *N* = XXX) = X.XX, *p* = .XXX, *V* = .XX

**예시:**

> 성별과 선호하는 학습 양식 간에 유의한 연관이 있었다, *χ*²(2, *N* = 180) = 8.74, *p* = .013, *V* = .22.

카이제곱 결과에는 항상 전체 표본 크기 *N*을 포함하십시오. 검정 통계량이 표본 크기에 직접 영향을 받기 때문입니다.

### Pearson 상관

**템플릿:** *r*(df) = .XX, *p* = .XXX

**예시:**

> 공부 시간과 시험 성적 간에 강한 정적 상관이 있었다, *r*(88) = .62, *p* < .001.

상관의 자유도는 *N* - 2입니다. *r*은 -1에서 1까지의 범위이므로 소수점 앞 0을 사용하지 않습니다. 설명된 분산을 전달하려면 *r*²를 보고합니다: *r*² = .38, 즉 공부 시간이 시험 점수 분산의 38%를 설명합니다.

### 단순 및 다중 회귀분석

**전체 모형 템플릿:** *F*(df1, df2) = X.XX, *p* = .XXX, *R*² = .XX

**개별 예측변인 템플릿:** *β* = X.XX, *t*(df) = X.XX, *p* = .XXX

**예시:**

> 회귀 모형은 통계적으로 유의하였다, *F*(3, 96) = 14.82, *p* < .001, *R*² = .32. 연습 시간(*β* = .41, *t*(96) = 4.23, *p* < .001)과 경력 년수(*β* = .28, *t*(96) = 2.87, *p* = .005)가 유의한 예측변인이었으나, 연령은 유의하지 않았다(*β* = .09, *t*(96) = 0.91, *p* = .365).

상대적 예측변인 중요성을 비교하려면 표준화 계수(*β*)를, 실질적 해석을 위해서는 비표준화 계수(*B*)를 보고합니다.

### Mann-Whitney U 검정

**템플릿:** *U* = XXX, *z* = X.XX, *p* = .XXX, *r* = .XX

**예시:**

> 치료 집단(*Mdn* = 4.5)의 만족도 평정이 통제 집단(*Mdn* = 3.0)보다 유의하게 높았다, *U* = 245, *z* = -2.89, *p* = .004, *r* = .36.

Mann-Whitney U의 기술통계로는 평균과 표준편차가 아닌 중앙값과 사분위범위를 보고합니다. 효과크기 *r*은 *z* / sqrt(*N*)으로 계산합니다.

## 유의한 결과 vs. 비유의한 결과

### 유의한 결과 보고(*p* < .05)

p값이 사전에 설정한 유의수준(일반적으로 .05) 미만인 경우 결과를 "통계적으로 유의한" 것으로 보고합니다. 정확한 p값을 사용하고 효과크기와 짝을 이루십시오.

> 차이는 통계적으로 유의하였다, *t*(58) = 2.41, *p* = .019, *d* = 0.63.

발견을 과장하는 언어를 피하십시오. "가설이 증명되었다"나 "치료가 확실히 효과가 있다"고 쓰지 마십시오. 통계적 유의성은 관찰된 데이터가 귀무가설 하에서 가능성이 낮다는 것을 의미합니다. 대립가설이 참이라는 것을 증명하지는 않습니다.

### 비유의한 결과 보고(*p* > .05)

p값이 alpha를 초과하면 결과를 "통계적으로 유의하지 않은" 것으로 보고합니다. 여전히 정확한 p값과 효과크기를 보고합니다.

> 차이는 통계적으로 유의하지 않았다, *t*(58) = 1.14, *p* = .259, *d* = 0.30.

**핵심 용어:** 비유의한 결과를 "무의미한(insignificant)"이라고 기술하지 마십시오. 일상 영어에서 "insignificant"는 "중요하지 않은"을 의미하며, 이는 통계적 비유의성이 의미하는 바가 아닙니다. 비유의한 결과는 연구가 검출할 검정력이 부족한 실재하지만 작은 효과를 반영할 수 있습니다.

또한 피해야 할 표현:
- "결과가 귀무가설을 증명했다"(귀무를 증명할 수 없음)
- "효과가 없었다"(증거의 부재가 부재의 증거는 아님)
- "집단이 동일했다"(비유의성은 동등성을 입증하지 않음)

대신: "통계적으로 유의한 차이가 없었다" 또는 "결과가 통계적 유의성에 도달하지 않았다"로 작성하십시오.

### 한계적으로 유의한 결과(*p* = .05~.10)

*p* 값이 .05와 .10 사이인 결과는 논쟁적 영역을 차지합니다. 일부 연구자는 이를 "한계적으로 유의" 또는 "유의성에 접근"이라고 기술합니다. 이 관행은 논란이 있습니다.

**"한계적으로 유의"에 반대하는 입장:** .05 기준은 임의적입니다. *p* = .06을 "한계적으로 유의"라고, *p* = .049를 "유의"라고 부르는 것은 존재하지 않는 의미 있는 경계를 암시합니다. 많은 방법론자와 편집자는 이 언어를 가설에 대한 지지의 외관을 부풀리는 일종의 소프트 p-해킹으로 봅니다.

**추세를 인정하는 입장:** 소표본 연구에서 *p* = .07이고 중간 크기의 효과가 있는 결과는 더 큰 표본이 필요한 실재하는 효과를 반영할 수 있습니다. 이러한 결과를 완전히 무시하는 것은 낭비적일 수 있습니다.

**권장 접근:** 정확한 p값, 효과크기, 신뢰구간을 보고합니다. 주관적 라벨을 붙이지 않고 데이터가 스스로 말하게 하십시오.

> 차이는 통계적 유의성에 도달하지 않았다, *t*(28) = 1.89, *p* = .069, *d* = 0.71, 95% CI [-0.05, 1.46].

효과크기(*d* = 0.71)와 0을 거의 배제하는 신뢰구간이 p값만으로는 알 수 없는 더 풍부한 이야기를 전달하는 것에 주목하십시오.

### 항상 효과크기를 포함

APA 7판(Section 6.36)은 명확합니다: "독자가 연구 결과의 크기 또는 중요성을 평가하기 위해, 어떤 형태의 효과크기 측정치를 포함하는 것이 거의 항상 필요하다."

유의 여부와 관계없이 모든 추론적 검정에 적절한 효과크기를 보고하십시오:

| 검정 | 효과크기 | 작은 효과 | 중간 효과 | 큰 효과 |
|------|---------|----------|----------|---------|
| t-검정 | Cohen's *d* | 0.20 | 0.50 | 0.80 |
| ANOVA | 편 *η*² | .01 | .06 | .14 |
| 카이제곱 | Cramér's *V* | .10 | .30 | .50 |
| 상관 | *r* | .10 | .30 | .50 |
| 회귀 | *R*² | .02 | .13 | .26 |
| Mann-Whitney U | *r* | .10 | .30 | .50 |

## 단측 vs. 양측 p값

### 사용 시기

**양측 검정**은 효과가 양 방향 중 어느 쪽으로든 존재하는지를 평가합니다. 사실상 모든 연구 맥락에서 기본값입니다.

**단측 검정**은 효과가 특정한 사전 지정된 방향으로 존재하는지를 평가합니다. 통계적 검정력이 더 높지만(필요 표본 크기가 약 20-25% 감소) 예측된 방향의 효과만 검출할 수 있습니다.

### APA에서의 보고 방법

단측 검정을 사용할 때는 이를 명시적으로 언급하고 방향적 예측을 정당화해야 합니다.

**양측(기본, 별도 표기 불필요):**

> *t*(58) = 2.41, *p* = .019, *d* = 0.63.

**단측(반드시 선언):**

> 중재가 읽기 속도를 증가시킨다는 이전 연구 결과에 근거하여 단측 검정을 사용하였다. 중재 집단이 유의하게 더 빠르게 읽었다, *t*(58) = 1.82, *p* = .037(단측), *d* = 0.47.

### 대부분의 학술지가 기대하는 것

대다수의 학술지는 데이터 수집 *전에* 방향적 가설에 대한 강력한 이론적 정당화를 제공하지 않는 한 양측 검정을 기대합니다. 비유의한 결과를 "구조"하기 위해 결과를 확인한 후 양측에서 단측 검정으로 전환하는 것은 p-해킹의 한 형태이며 연구 윤리를 위반합니다.

단측 검정을 사전등록했다면 사전등록을 인용하십시오. 사전등록하지 않았다면 양측 검정을 사용하십시오.

## 흔한 실수와 주의사항

### 1. *p* = .000 작성

가장 흔한 p값 오류입니다. SPSS, Excel 및 많은 프로그램이 소수점 세 자리에서 반올림하면 0이 되는 경우 .000을 표시합니다. 실제 p값은 매우 작은 양수입니다. *p* < .001로 쓰십시오.

### 2. 소수점 앞 0 추가

*p* = .042 대신 *p* = 0.042로 쓰는 것은 APA 양식을 위반합니다. 절대값이 1.0을 초과할 수 없는 통계량에는 소수점 앞 0을 생략합니다.

### 3. 정확한 p값 미보고

정확한 값 없이 *p* < .05만 쓰는 것은 이전 APA 판에서는 허용되었지만 더 이상 충분하지 않습니다. APA 7판은 정확한 값을 요구합니다(예: *p* = .032). 유일한 예외는 *p* < .001인 경우입니다.

### 4. 가설이 "증명되었다"고 말하기

통계 검정은 가설을 증명하지 않습니다. 관찰된 데이터가 귀무가설과 일관적인지를 평가합니다. "결과가 가설을 지지했다" 또는 "귀무가설을 기각했다"로 쓰고, "가설이 증명되었다"로 쓰지 마십시오.

### 5. 유의성 라벨만 보고

검정 통계량, 자유도, 정확한 p값, 효과크기를 제공하지 않고 "결과가 유의했다"라고만 쓰면 독자가 증거를 평가할 수 없습니다. 모든 유의성 주장에는 완전한 통계적 출력이 포함되어야 합니다.

### 6. *p*를 이탤릭체로 쓰지 않음

APA 원고에서 *p*는 항상 이탤릭체입니다. 원고 본문, 표, 그림 모두에 적용됩니다. 대부분의 학술지가 이를 엄격히 시행합니다.

### 7. 통계적 유의성과 실질적 중요성의 혼동

*p* = .001이고 *d* = 0.08인 결과는 통계적으로 유의하지만 실질적으로 사소합니다. 반대로, *p* = .08이고 *d* = 0.75인 결과는 통계적으로 유의하지 않지만 검정력이 부족했을 가능성이 있는 상당한 효과를 시사합니다. 항상 효과크기와 함께 p값을 해석하십시오.

### 8. 효과크기 미보고

APA 7판은 모든 추론적 검정에 효과크기를 의무화합니다. 이를 생략하면 양식 위반이며, 더 중요하게는 독자가 실질적 유의성을 평가하는 데 필요한 정보를 박탈합니다.

### 9. 정당화 없이 "한계적으로 유의" 사용

결과를 "한계적으로 유의"하다고 기술하면 심사자가 이의를 제기할 준비를 하십시오. 많은 편집자가 이 언어를 수용 불가로 간주합니다. 사용하는 경우 효과크기, 신뢰구간, 명확한 근거를 제공하십시오.

### 10. "유의성에 접근"이라고 쓰기

"한계적으로 유의"보다 더 문제적인 표현입니다. p값은 아무 것에도 "접근"하지 않습니다. 값은 있는 그대로입니다. 정확한 숫자를 보고하고 독자가 판단하도록 하십시오.

## p값 기준: Alpha = .05, .01, .001

### 각 기준의 의미

유의수준(alpha, *α*)은 데이터를 분석하기 *전에* 설정하는 통계적 유의성의 기준입니다.

- ***α* = .05**는 제1종 오류(참인 귀무가설을 거짓으로 기각)의 5% 확률을 수용한다는 것입니다. 대부분의 사회과학 및 행동과학에서 표준입니다.
- ***α* = .01**은 1% 확률을 수용합니다. 더 보수적인 증거가 필요한 분야나 상황에서 사용됩니다.
- ***α* = .001**은 0.1% 확률을 수용합니다. 유전체 전체 연관 연구나 입자물리학 등 매우 높은 증거 기준이 적용되는 맥락에서 사용됩니다.

### 더 엄격한 기준을 사용하는 경우

.05보다 엄격한 기준을 고려해야 하는 상황:

**다중 비교.** 동일한 데이터에서 여러 검정을 수행하면 최소 하나의 거짓 양성 확률이 증가합니다. Bonferroni 보정은 alpha를 비교 횟수로 나눕니다. 10개의 검정을 수행하면 각 개별 비교는 *α* = .005를 사용합니다.

**고위험 의사결정.** 약물 승인을 위한 임상시험, 안전이 중요한 공학적 결정, 또는 주요 정책적 함의가 있는 연구는 *α* = .01 이상을 정당화할 수 있습니다.

**재현 우려.** Benjamin et al.(2018)은 새로운 발견 주장에 대해 *p* < .005를 통계적 유의성의 새로운 기준으로, .005와 .05 사이의 결과에는 "시사적(suggestive)"이라는 용어를 제안하였습니다. 이 제안은 여전히 논의 중이지만 출판된 연구의 거짓 양성률에 대한 증가하는 우려를 반영합니다.

**탐색적 분석.** 사전등록되지 않은 분석의 경우 더 엄격한 기준이 데이터 주도적 발견에 대한 부분적 안전장치를 제공합니다.

### 기준 보고 방법

방법 섹션에서 유의수준을 명시하고 일관되게 적용하십시오.

> 모든 분석에서 *α* = .05에서 통계적 유의성을 평가하였다. 사후 쌍별 비교에는 Bonferroni 보정 alpha .017(.05 / 3개 비교)을 사용하였다.

## APA 7판의 p값에 대한 입장

### p값만으로는 불충분

APA Publication Manual(Section 6.36)은 p값이 연구 결과의 유일한 지표가 되어서는 안 된다고 명시합니다:

> "독자가 연구 결과의 크기 또는 중요성을 평가하기 위해, 결과 섹션에 어떤 형태의 효과크기 측정치를 포함하는 것이 거의 항상 필요하다."

이 지침은 영가설 유의성 검정(NHST)에 대한 수십 년간의 방법론적 비판을 반영합니다. p값은 효과가 0이 아닌 가능성이 있는지를 알려주지만, 효과가 얼마나 크거나 중요한지에 대해서는 아무것도 말하지 않습니다.

### 효과크기와 신뢰구간이 필수

APA 7판은 다음을 보고하도록 권장합니다:

1. 모든 추론적 검정에 대한 **효과크기**(Cohen's *d*, *η*², *r*, *R*², Cramér's *V*, 오즈비 등)
2. 효과크기와 모수 추정에 대한 **신뢰구간**(일반적으로 95% 수준)

잘 보고된 결과의 예:

> 치료 집단이 통제 집단보다 유의하게 더 많이 향상되었다, *t*(78) = 3.12, *p* = .003, *d* = 0.70, 95% CI [0.24, 1.15].

이 문장은 독자에게 모든 것을 제공합니다: 검정 통계량, 귀무에 대한 증거(*p* = .003), 효과의 크기(*d* = 0.70, 중간~큰 효과), 그리고 추정의 정밀도(진정한 효과는 0.24에서 1.15 표준편차 사이).

### 통계적 유의성에 대한 지속적인 논의

학계는 전환기에 있습니다. 현재 사고를 형성한 몇 가지 발전:

- **ASA의 p값에 관한 성명(2016)**은 p < .05를 이분법적 결정 규칙으로 사용하는 것에 대해 경고하고 p값이 가설이 참일 확률을 측정하지 않음을 강조하였습니다.
- **"통계적 유의성 재정의" 논문(Benjamin et al., 2018)**은 새로운 발견 주장에 *p* < .005를 제안하였습니다.
- ***Nature*의 "통계적 유의성 은퇴" 논설(Amrhein et al., 2019)**에는 800명 이상의 과학자가 서명하여 통계적 유의성 개념 자체의 폐지를 주장하였습니다.
- **APA 7판(2020)**은 중도적 입장을 취합니다: p값을 계속 보고하되 항상 효과크기와 신뢰구간을 함께 제시합니다. .05를 엄격한 경계선으로 취급하지 마십시오.

실제로 이는 모든 분석에 대해 정확한 p값, 효과크기, 신뢰구간을 보고해야 함을 의미합니다. 단일 기준에 의존하기보다 독자가 증거의 전체상을 평가하도록 하십시오.

## 제출 전 체크리스트

제출 전, 논문의 모든 p값에 대해 이 체크리스트를 확인하십시오:

- *p*가 이탤릭체인가?
- = 또는 < 앞뒤에 공백이 있는가?
- 소수점 이하 세 자리까지 보고했는가?
- *p* = .000 대신 *p* < .001로 썼는가?
- 소수점 앞에 0이 없는가?
- 정확한 p값을 보고했는가(*p* < .05만이 아닌)?
- p값과 함께 효과크기를 보고했는가?
- 기술통계(평균, SD 또는 중앙값)를 포함했는가?
- 검정 통계량에 자유도를 포함했는가?
- "증명됨", "무의미한", "유의성에 접근" 같은 언어를 피했는가?

모든 항목을 확인했다면, 귀하의 p값 보고는 APA 7판 기준에 부합합니다.

## p값을 자동으로 서식화하는 도구

수작업으로 p값을 서식화하는 것은 번거롭고 오류가 발생하기 쉽습니다. 여러 도구가 도움이 될 수 있습니다:

- **StatMate**([statmate.org](https://statmate.org))는 t-검정, ANOVA, 카이제곱, 상관, 회귀분석 등 20개 통계 검정에 대해 APA 형식의 결과 문장을 자동으로 생성합니다. 데이터를 입력하면 계산기가 올바른 p값 서식, 효과크기, 신뢰구간이 포함된 출판 준비된 문장을 산출합니다. 결과를 논문에 바로 복사할 수 있습니다.
- **R**의 `papaja` 패키지에 있는 `apa_print()` 함수가 서식화된 통계량을 출력합니다.
- **JASP**와 **jamovi**는 기본 출력에 APA 형식의 표를 포함합니다.
- **SPSS**는 p값을 APA 양식으로 서식화하지 않습니다. "Sig. = .000"을 *p* < .001로 수동 변환하고 소수점 앞 0을 제거해야 합니다.

## 요약

APA 7판 형식으로 p값을 보고하려면 다섯 가지가 필요합니다: 소수점 이하 세 자리까지의 정확한 값, 매우 작은 값의 경우 *p* < .001, 소수점 앞 0 없음, 이탤릭체 *p*, 연산자 앞뒤 공백. 서식을 넘어, APA는 p값이 항상 효과크기와 이상적으로는 신뢰구간을 동반해야 한다고 강조합니다.

가장 중요한 원칙은 다음입니다: p값은 증거의 한 부분이지 전체 이야기가 아닙니다. 적절하게 보고된 결과에는 기술통계, 자유도가 포함된 검정 통계량, 정확한 p값, 효과크기, 그리고 신뢰구간이 포함됩니다. 이들이 함께 독자에게 연구 결과를 평가하는 데 필요한 모든 것을 제공합니다.
