## 回帰分析の正しい報告が重要な理由

回帰分析は、定量的研究において最も汎用性の高い手法の一つです。学習時間から試験成績を予測する場合でも、複数の人口統計学的変数が職務満足度に与える複合的な影響をモデル化する場合でも、回帰分析がその枠組みを提供します。しかし、回帰分析の結果をAPA形式で報告する段階で、多くの研究者がつまずきます。

t検定のような単純な検定とは異なり、回帰分析の報告には複数の情報層が含まれます。モデル全体の適合度、個々の予測変数の寄与、標準化係数と非標準化係数、信頼区間などです。これらの要素のいずれかを省略すると、ジャーナルの査読者から修正を求められることになります。本ガイドでは、単回帰分析と重回帰分析の両方について、APA第7版の報告形式を具体的な数値例とともに解説します。

## 単回帰分析

### 必須要素

APA形式で報告するすべての単回帰分析の結果には、以下を含める必要があります。

- **決定係数（*R*²）**：予測変数によって説明される結果変数の分散の割合
- **F統計量**：モデル全体の有意性検定、イタリック体で*F*と表記
- **自由度**：回帰の自由度と残差の自由度を括弧内に記載
- **正確な*p*値**：小数点以下3桁まで
- **非標準化係数（*B*）**：傾き、標準誤差付き
- **標準化係数（*β*）**：*β*として報告

モデル全体の基本テンプレート：

> *R*² = .XX, *F*(df_回帰, df_残差) = X.XX, *p* = .XXX

### 研究場面

ある研究者が、120名の大学生を対象に、1週間あたりの学習時間が期末試験の成績を予測するかどうかを調べています。

### モデル全体の報告

**APA報告例：**

> 週間学習時間が期末試験の成績を予測するかどうかを調べるために単回帰分析を実施した。その結果、モデル全体は統計的に有意であった, *R*² = .34, *F*(1, 118) = 60.73, *p* < .001。週間学習時間は期末試験成績の分散の34%を説明した。

### 係数の報告

> 週間学習時間は期末試験の成績を有意に予測した, *B* = 2.85, *SE* = 0.37, *t*(118) = 7.79, *p* < .001, 95% CI [2.12, 3.58]。週間学習時間が1時間増加するごとに、期末試験の成績は平均2.85ポイント上昇した。

### 各要素の解説

| 要素 | 値 | 説明 |
|------|-----|------|
| *R*² | .34 | 説明された分散の割合34%、先頭のゼロなし |
| *F* | 60.73 | モデル全体のF統計量、小数点以下2桁 |
| df | 1, 118 | 回帰の自由度（予測変数の数）と残差の自由度（*N* - *k* - 1） |
| *p* | < .001 | 正確なp値、非常に小さい場合は< .001 |
| *B* | 2.85 | 非標準化傾き |
| *SE* | 0.37 | 傾きの標準誤差 |
| *t* | 7.79 | 係数のt統計量 |
| 95% CI | [2.12, 3.58] | *B*の信頼区間 |

## 重回帰分析

### 複数の予測変数がある場合

重回帰分析は、2つ以上の予測変数を含めることで単回帰モデルを拡張します。各予測変数の固有の寄与に加えてモデル全体を報告する必要があるため、APA報告はより複雑になります。

### 研究場面

ある研究者が、150名の学生を対象に、学習時間、授業出席率、高校GPAが期末試験の成績を予測するかどうかを調査しています。

### モデル全体の報告

> 学習時間、授業出席率、高校GPAが期末試験の成績を予測するかどうかを調べるために重回帰分析を実施した。モデル全体は統計的に有意であった, *R*² = .52, 調整済み*R*² = .51, *F*(3, 146) = 52.78, *p* < .001。3つの予測変数は合わせて期末試験成績の分散の52%を説明した。

注：重回帰分析では、*R*²と**調整済み*R*²**の両方を必ず報告してください。調整済み*R*²はモデル内の予測変数の数を補正し、特に予測変数の数がサンプルサイズに対して大きい場合に、説明された分散のより正確な推定値を提供します。

### 係数表

個々の予測変数の結果は、表にまとめると明確になります。

| 予測変数 | *B* | *SE* | *β* | *t* | *p* | 95% CI |
|---------|------|------|------|------|------|---------|
| （切片） | 12.45 | 5.32 | -- | 2.34 | .021 | [1.94, 22.96] |
| 学習時間 | 1.92 | 0.31 | .38 | 6.19 | < .001 | [1.31, 2.53] |
| 出席率 | 0.28 | 0.08 | .22 | 3.50 | < .001 | [0.12, 0.44] |
| 高校GPA | 8.74 | 1.85 | .29 | 4.72 | < .001 | [5.08, 12.40] |

### 個々の予測変数の報告

> 学習時間が期末試験成績の最も強い予測変数であった, *B* = 1.92, *SE* = 0.31, *β* = .38, *t*(146) = 6.19, *p* < .001, 95% CI [1.31, 2.53]。高校GPAも試験成績を有意に予測した, *B* = 8.74, *SE* = 1.85, *β* = .29, *t*(146) = 4.72, *p* < .001, 95% CI [5.08, 12.40]。授業出席率はより小さいが統計的に有意な寄与を示した, *B* = 0.28, *SE* = 0.08, *β* = .22, *t*(146) = 3.50, *p* < .001, 95% CI [0.12, 0.44]。

### 標準化ベータの理解

標準化係数（*β*）により、同じモデル内での予測変数の重要性を直接比較することが可能になります。*β*値は元の測定単位ではなく標準偏差の単位で表されるため、尺度の違いにかかわらず予測変数間で比較できます。

上記の例では、学習時間の*β* = .38に対して出席率の*β* = .22であることから、他の変数を一定に保った場合、学習時間が試験成績の予測に比較的強い固有の寄与を持つことがわかります。

## 非有意な結果

非有意な回帰分析の結果も同じ形式に従います。有意でない場合でも、モデル全体の統計量と個々の係数を必ず報告してください。

> 単回帰分析の結果、睡眠時間は試験成績を有意に予測しなかった, *R*² = .02, *F*(1, 118) = 2.41, *p* = .123。睡眠時間は有意な予測変数ではなかった, *B* = 0.95, *SE* = 0.61, *t*(118) = 1.55, *p* = .123, 95% CI [-0.26, 2.16]。

## よくある間違い

### 重回帰分析での調整済みR²の省略

単回帰分析では*R*²のみで十分ですが、重回帰分析で調整済み*R*²を省略するのはよくある見落としです。調整済み*R*²はモデルを有意に改善しない予測変数の追加に対してペナルティを課すため、複数の予測変数がある場合のモデル適合度の評価に不可欠です。

### BとBetaの混同

*B*（非標準化）と*β*（標準化）は異なる情報を伝えます。*B*は、予測変数が元の単位で1単位変化したときの結果変数の変化を示します。*β*は標準偏差の単位での変化を示し、予測変数間の比較を可能にします。どちらを報告しているかを常に明示し、可能であれば両方を含めてください。

### p = .000の報告

統計ソフトウェアがp = .000と出力することがあります。これは確率が正確にゼロであることを意味しません。非常に小さいp値は必ず*p* < .001として報告してください。

### 信頼区間の欠落

APA第7版では、回帰係数の95%信頼区間を報告することを強く推奨しています。信頼区間は、p値だけでは伝えられない推定値の精度に関する情報を提供します。これを含めることは、厳密な報告実践を示すものです。

### 切片の報告忘れ

切片は通常、解釈の焦点ではありませんが、係数表には含める必要があります。査読者は切片の行を含む完全な表を期待しています。

### 仮定の確認を怠る

回帰分析の結果を報告する前に、主要な仮定を確認してください。線形性、残差の独立性、等分散性（残差の分散が一定であること）、残差の正規性、および多重共線性の不在（重回帰の場合）です。仮定の違反と講じた修正措置を結果セクションに記述してください。

## APA回帰分析報告チェックリスト

原稿を提出する前に、回帰分析の結果に以下が含まれていることを確認してください。

- モデル全体の*R*²（重回帰の場合は調整済み*R*²も）
- 正しい自由度を伴う*F*統計量
- モデル全体の正確な*p*値
- 各予測変数の*B*、*SE*、*β*、*t*、*p*、95% CIを含む係数表
- 係数表の切片行
- 非標準化係数（*B*）と標準化係数（*β*）を区別する明確なラベル
- すべての統計記号（*R*²、*F*、*B*、*β*、*t*、*p*）のイタリック体表記
- 本文中での仮定の確認への言及

## 重回帰分析をAPA形式で報告する

### 単回帰との報告上の違い

単回帰では予測変数が1つであるため、モデル全体の検定と係数の検定は同じ結果を返します。一方、重回帰では2つの層の報告が必要です。第一に、モデル全体の*F*検定は、予測変数のセット全体が有意な分散を説明しているかどうかを示します。第二に、各予測変数は他の予測変数を統制した後の固有の寄与を示す個別の*t*検定、標準誤差、信頼区間を持ちます。

予測変数間で分散が共有されるため、個々の*R*²値の合計はモデル全体の*R*²を超えるのが一般的です。これが重回帰で調整済み*R*²を報告する理由であり、個々の予測変数を解釈する際に固有の寄与と共有された寄与を区別する表現が必要な理由です。

### 予測変数2つのAPA報告例

週間運動時間と睡眠の質が、社員100名のストレスレベルを予測するかどうかを検証する研究を想定します。

> 週間運動時間と睡眠の質が知覚ストレスレベルを予測するかどうかを調べるために重回帰分析を実施した。モデル全体は統計的に有意であった, *R*² = .28, 調整済み*R*² = .27, *F*(2, 97) = 18.45, *p* < .001。2つの予測変数は知覚ストレスの分散の28%を説明した。

### 個々の予測変数の報告

各予測変数について、非標準化係数*B*、標準誤差、標準化係数*β*、残差自由度を含む*t*統計量、*p*値、95%信頼区間を報告します。

> 週間運動時間はストレスレベルを有意に予測した, *B* = -1.47, *SE* = 0.38, *β* = -.34, *t*(97) = -3.87, *p* < .001, 95% CI [-2.22, -0.72]。睡眠の質を統制した状態で、週間運動時間が1時間増加するごとに知覚ストレスは1.47ポイント低下した。

> 睡眠の質もストレスレベルを有意に予測した, *B* = -2.83, *SE* = 0.71, *β* = -.35, *t*(97) = -3.99, *p* < .001, 95% CI [-4.24, -1.42]。運動時間を統制した状態で、睡眠の質が1単位向上するごとに知覚ストレスは2.83ポイント低下した。

### R²変化量（Delta R²）の解釈

階層的回帰分析で予測変数を段階的に投入する場合、*R*²変化量（*Delta R*²）を報告して、各段階で追加的に説明される分散の割合を示します。

> 第1ステップで週間運動時間を投入し、ストレスの分散の15%を説明した, *R*² = .15, *F*(1, 98) = 17.29, *p* < .001。第2ステップで睡眠の質を追加投入し、モデルはストレスの分散を追加で13%説明した, *Delta R*² = .13, *F*-change(1, 97) = 19.12, *p* < .001。

この方法は、統制変数を先に投入し主要な予測変数を後から投入するなど、理論的根拠に基づいて投入順序を決める場合に有用です。

## 回帰分析の仮定をAPAで報告する

### 5つの重要な仮定

回帰分析の結果を提示する前に、以下の仮定を検証し、結果セクションで簡潔に言及する必要があります。

**1. 線形性。** 各予測変数と結果変数の関係が線形である必要があります。散布図や偏回帰プロットを検討してください。非線形性が認められる場合は、変数変換や多項式項の追加を検討します。

**2. 残差の独立性。** 残差同士が相関してはなりません。この仮定は縦断データやクラスターデータで特に重要です。Durbin-Watson統計量を報告します：

> Durbin-Watson統計量は1.92であり、残差間に実質的な自己相関がないことを示した（2.0に近い値は独立した誤差を示唆する）。

**3. 等分散性（Homoscedasticity）。** 残差の分散がすべての予測値にわたって一定である必要があります。残差対予測値のプロットを検討してください。漏斗状のパターンが見られる場合、不等分散性が存在する可能性があります。

> 標準化残差と標準化予測値のプロットはランダムな散布パターンを示し、等分散性の仮定が満たされていることが示唆された。

**4. 残差の正規性。** 残差がおおよそ正規分布に従う必要があります。残差のヒストグラムやQ-Qプロットを検討し、必要に応じてShapiro-Wilk検定を実施します。

> 標準化残差のヒストグラムとQ-Qプロットを検討した結果、おおよそ正規分布に従うことが確認された。

**5. 多重共線性（重回帰のみ）。** 予測変数同士が過度に高い相関を持ってはなりません。各予測変数の分散拡大係数（VIF）を報告します：

> すべての予測変数のVIF値は1.12から2.34の範囲であり、一般的に使用される閾値10を大幅に下回っており、多重共線性の問題がないことが確認された。

### 仮定が違反された場合の報告方法

仮定が違反された場合、違反の内容と講じた修正措置を記述します。主な戦略は以下の通りです。

- **非線形性**：対数変換や平方根変換の適用、多項式項の追加
- **自己相関**（Durbin-Watsonが2.0から大きく外れる）：一般化最小二乗法の使用、ラグ変数の追加
- **不等分散性**：不均一分散に頑健な標準誤差（HC3等）や加重最小二乗法の使用
- **残差の非正規性**：変数変換やブートストラップ信頼区間の使用
- **多重共線性**（VIF 10超）：相関の高い予測変数の除去・統合、リッジ回帰の使用

違反を報告する例：

> 残差プロットの視覚的検討の結果、不等分散性が存在する可能性が確認された。これに対処するため、すべての係数検定に不均一分散に頑健な標準誤差（HC3）を適用した。頑健な標準誤差を適用した後も、有意な予測変数のパターンは変わらなかった。

## 回帰係数の信頼区間

### APA第7版が信頼区間を推奨する理由

APA第7版が信頼区間を重視するのは、信頼区間が点推定値と推定の精度という2つの情報を同時に伝えるためです。狭いCIは精密な推定を、広いCIは不確実性を示唆します。さらに、CIは係数がゼロと統計的に異なるかどうかを直接示します：95% CIがゼロを含まなければ、その予測変数は.05水準で有意です。

### 信頼区間を含む報告例

> 週間学習時間は期末試験の成績を有意に予測した, *B* = 2.34, 95% CI [0.89, 3.79], *t*(118) = 3.21, *p* = .002。信頼区間は、母集団における真の傾きが学習1時間あたり0.89ポイントから3.79ポイントの間にあると推定されることを示す。

### 信頼区間の解釈方法

- **CIがゼロを含まない**：対応するアルファ水準で係数が統計的に有意
- **CIがゼロを含む**：係数が統計的に有意でない
- **狭いCI**：推定が精密（大標本や低い変動性による場合が多い）
- **広いCI**：推定が不精密（より大きな標本やより良い測定が必要）

### 複数の予測変数に対する完全なAPA例

> 学習時間は試験成績を有意に予測した, *B* = 1.92, 95% CI [1.31, 2.53], *β* = .38, *t*(146) = 6.19, *p* < .001。高校GPAも有意な予測変数であった, *B* = 8.74, 95% CI [5.08, 12.40], *β* = .29, *t*(146) = 4.72, *p* < .001。出席率はより小さいが有意な寄与を示した, *B* = 0.28, 95% CI [0.12, 0.44], *β* = .22, *t*(146) = 3.50, *p* < .001。

3つの予測変数すべての信頼区間がゼロを含んでおらず、これは3変数すべてが統計的に有意であることと一致します。

## 標準化係数 vs. 非標準化係数

### BとBeta、どちらを報告すべきか

非標準化係数（*B*）と標準化係数（*β*）はそれぞれ異なる目的を持ち、APAは可能な限り両方の報告を推奨しています。

### 非標準化係数（B）

*B*は、予測変数が元の単位で1単位変化したときの結果変数の予測される変化量を示します。元の測定単位が保持されるため、実践的な解釈に適しています。

> 週間学習時間が1時間増加するごとに、期末試験の成績は2.85ポイント上昇した（*B* = 2.85）。

読者は時間と試験点数という意味のある単位で効果の大きさを即座に理解できます。これは、年、円、kgなど直感的な測定単位を使用する場合に特に有用です。

### 標準化係数（Beta）

*β*は、予測変数が1標準偏差変化したときの結果変数の変化量を標準偏差単位で示します。元の尺度が除去されるため、モデル内での予測変数の相対的重要性を直接比較できます。

> 学習時間の相対的寄与（*β* = .38）は出席率（*β* = .22）より試験成績の予測において強かった。

異なる尺度で測定された予測変数の相対的強さを比較する際に*β*を使用します。例えば、時間（0〜40）と出席率（0〜100%）の比較は、標準化係数を通じてのみ意味があります。

### APA形式で両方の係数を報告する

> 学習時間は試験成績を有意に予測した, *B* = 1.92, *SE* = 0.31, *β* = .38, *t*(146) = 6.19, *p* < .001, 95% CI [1.31, 2.53]。

この1文に、実践的解釈のための*B*と相対的比較のための*β*の両方が含まれ、標準誤差、検定統計量、有意確率、信頼区間まで提示されています。

## 回帰分析結果のAPA表形式

### 表と本文記述の使い分け

予測変数が1つの単回帰では本文記述で十分です。予測変数が2つ以上の場合は表を使用してください。3つ以上の予測変数がある場合、本文のみでの報告は読みにくくなるため、表が不可欠です。

### 完全なAPA回帰係数表

適切に構成されたAPA回帰結果表の列構成は以下の通りです。

| 変数 | *B* | *SE* | *β* | *t* | *p* | 95% CI |
|------|------|------|------|------|------|---------|
| （切片） | 12.45 | 5.32 | -- | 2.34 | .021 | [1.94, 22.96] |
| 学習時間 | 1.92 | 0.31 | .38 | 6.19 | < .001 | [1.31, 2.53] |
| 出席率 | 0.28 | 0.08 | .22 | 3.50 | < .001 | [0.12, 0.44] |
| 高校GPA | 8.74 | 1.85 | .29 | 4.72 | < .001 | [5.08, 12.40] |

表の下にモデル要約の注を追加します：

> *注.* *R*² = .52, 調整済み*R*² = .51, *F*(3, 146) = 52.78, *p* < .001。

### 表作成のガイドライン

- APA表形式を使用し、ヘッダー行の上下と表の下部に水平線を配置します
- APA表では垂直線を使用しません
- 列ヘッダーのすべての統計記号をイタリック体で表記します：*B*、*SE*、*β*、*t*、*p*
- 切片を最初の行に配置し、その後に予測変数を理論的重要度または投入順序に従って並べます
- *B*と*SE*は小数点以下2桁まで、*β*は先頭のゼロなしで小数点以下2桁まで報告します
- 表番号とタイトルを含めます（例：「表1. 期末試験成績を予測する重回帰分析結果」）

### モデル要約行

一部のジャーナルでは、係数表の上または下に別個のモデル要約セクションを好みます。含めるべき項目は以下の通りです。

- *R*²と調整済み*R*²
- 自由度を含むモデル全体の*F*統計量
- モデルの*p*値
- 推定の標準誤差（任意だが情報価値が高い）

## よくある質問（FAQ）

### R²と調整済みR²の違いは何ですか？

*R*²はすべての予測変数が合わせて説明する分散の割合を示します。調整済み*R*²は予測変数の数を考慮して補正し、不必要な変数を含むモデルにペナルティを課します。予測変数の数が異なるモデルを比較する際には調整済み*R*²を使用してください。予測変数が1つの単回帰ではほぼ同じ値になりますが、重回帰では調整済み*R*²は常に*R*²と同じかそれ以下であり、意味のない予測変数が多いほどその差は大きくなります。

### 負の回帰係数はどう解釈しますか？

負の係数は、予測変数が1単位増加すると、他の予測変数を統制した状態で結果変数がその係数の値だけ減少することを意味します。符号は関係の方向を示すものであり、強さを示すものではありません。例えば、ストレスを予測する運動時間の*B* = -1.47は、運動1時間の増加がストレス1.47ポイントの減少と関連することを意味します。

### 標準化係数と非標準化係数のどちらを報告すべきですか？

APAは両方の報告を推奨しています。非標準化係数（*B*）は元の測定単位を保持し、実践的な解釈に適しています。例えば、学習1時間の追加が試験2.85点の上昇を予測することを読者が理解できます。標準化係数（*β*）はモデル内での予測変数の相対的重要性を比較可能にし、予測変数の尺度が異なる場合に不可欠です。

### VIFが10を超えるとどういう意味ですか？

分散拡大係数（VIF）が10を超えると、深刻な多重共線性が存在することを示し、2つ以上の予測変数が非常に高い相関を持つことを意味します。これにより標準誤差が膨張し、個々の係数検定の信頼性が低下し、係数の符号が逆転したり有意な効果が隠されたりする可能性があります。相関の高い予測変数の一方を除去するか、合成変数に統合するか、リッジ回帰を使用することを検討してください。

### カテゴリカル予測変数で回帰分析はできますか？

はい、カテゴリカル変数をダミー変数（0/1コーディング）に変換することで可能です。*k*個のカテゴリを持つ変数は*k* - 1個のダミー変数が必要です。すべてのダミー変数が0の場合が参照カテゴリとなります。例えば、3水準の教育変数（高卒、学士、大学院）は2つのダミー変数が必要で、1つの水準が参照群として機能します。

### モデルが有意でない場合、どう報告しますか？

有意なモデルと同じ統計量を報告します：*F*統計量、自由度、*p*値、*R*²。結果が有意でなくてもモデル全体と個々の係数を漏れなく報告してください。

> 回帰モデルは統計的に有意ではなかった, *F*(2, 97) = 1.45, *p* = .240, *R*² = .03。週間運動時間（*B* = -0.52, *p* = .187）も睡眠の質（*B* = -0.84, *p* = .291）もストレスレベルを有意に予測しなかった。

### 単回帰と重回帰の違いは何ですか？

単回帰は1つの予測変数で結果変数を説明し、重回帰は2つ以上の予測変数を同時に使用します。重回帰は各予測変数が説明する共有分散と固有分散を考慮するため、重回帰での個々の係数は別々の単回帰で得られた係数とは異なる場合があります。また、重回帰は調整済み*R*²を提供しますが、これは単回帰では不要です。

### 回帰係数の信頼区間を報告すべきですか？

はい。APA第7版は回帰係数の95%信頼区間の報告を推奨しています。信頼区間は推定の精度と、係数がゼロと有意に異なるかどうかの情報を提供します。95% CIがゼロを含まなければ、その係数は.05水準で統計的に有意です。さらに、信頼区間は読者が効果の妥当な範囲を把握し、実質的有意性を評価するのに役立ちます。

## StatMateでAPA形式の回帰分析結果を生成する

回帰分析の出力を正しくフォーマットすること、特に複数の予測変数を含む重回帰分析では、煩雑でエラーが起きやすい作業です。StatMateの回帰分析計算ツールがそのプロセス全体を代行します。

単回帰または重回帰計算ツールにデータを入力すると、StatMateが*R*²、調整済み*R*²、*F*統計量、標準誤差付きの個々の係数、標準化ベータ、t統計量、p値、信頼区間を自動的に計算します。結果はAPA第7版形式でフォーマットされ、原稿にそのままコピーできます。

StatMateに計算とフォーマットを任せることで、係数の入れ替え、信頼区間の欠落、自由度の間違いなどの一般的なエラーを排除し、結果の解釈と考察の強化に時間を費やすことができます。

## まとめ

回帰分析の結果をAPA形式で報告するには、モデル全体の適合度と個々の予測変数の寄与の両方を提示する必要があります。単回帰分析では、**R²、自由度を伴うF統計量、標準誤差・t統計量・p値・信頼区間付きの係数**を含めてください。重回帰分析では、さらに**調整済みR²、標準化ベータを含む完全な係数表**を追加し、各予測変数の相対的な重要性を強調する説明文を記述してください。回帰分析の結果を報告する際には、本ガイドのテンプレートとチェックリストを参考にしてください。
