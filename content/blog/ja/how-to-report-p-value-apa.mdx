## APAにおけるp値報告が重要な理由

すべての量的研究論文にはp値が含まれていますが、p値は学術論文で最も頻繁に誤って報告される統計量の一つです。ジャーナルは不正確な書式の原稿を却下します。査読者は *p* = .000と書いたり小数点の前にゼロを付けた論文にフラグを立てます。読者は統計的有意性と実践的重要性を混同する著者に対する信頼を失います。

米国心理学会は、*Publication Manual of the American Psychological Association, 7th Edition*（2020）において正確なルールを確立し、これらの問題に対処しました。これらのルールは恣意的なスタイルの好みではありません。行動科学、社会科学、健康科学全体で統計的証拠の明確さ、一貫性、正確なコミュニケーションを確保するために存在しています。

p値の報告を正しく行うことが重要な理由は3つあります：

1. **再現性。** 正確なp値を報告することで、他の研究者が有意/非有意という二値ラベルに頼るのではなく、自ら証拠の強さを評価できます。
2. **メタ分析。** 研究間の知見を統合する研究者には、正確なp値と効果量が必要です。*p* < .05のみの報告は、将来のメタ分析者が必要とする情報を破壊します。
3. **出版基準。** 心理学、教育学、看護学、公衆衛生の主要ジャーナルはAPA書式を施行しています。不正確なp値の報告は学術的慣例への不慣れを示し、出版を遅延させたり妨げたりする可能性があります。

本ガイドでは、APA第7版形式でのp値報告に関して知っておくべきすべてのルール、すべての一般的な検定、すべての間違いを網羅しています。

## p値の核心的APAルール

APA出版マニュアル（セクション6.36）は、p値の提示方法を正確に規定しています。5つのルールがあり、いずれも交渉の余地はありません。

### ルール1：正確なp値を小数点以下2桁または3桁で報告する

APA第7版は、小数点以下2桁または3桁に丸めた正確なp値を要求しています。3桁が最も一般的な慣例であり、最も多くの情報を提供します。

- 正しい：*p* = .032
- 正しい：*p* = .007
- 不正確：*p* = .03（不正確）
- 不正確：*p* < .05（正確でない）

正確な値を報告することで、読者が自ら証拠を評価できます。*p* = .048の結果は *p* = .002とは非常に異なる物語を伝えますが、どちらも「.05水準で有意」です。

### ルール2：値が.001未満の場合は *p* < .001を使用する

p値が.001より小さい場合、正確な値を報告しないでください。代わりに *p* < .001と書いてください。

- 正しい：*p* < .001
- 不正確：*p* = .000（確率は決して正確にゼロではない）
- 不正確：*p* = .0001（小数点以下の桁数が多すぎる）
- 不正確：*p* = 0.000（2つのエラー：先頭ゼロと不可能な値）

SPSSなどの統計ソフトウェアは、出力に「Sig. = .000」と表示することがあります。これは丸めのアーティファクトであり、実際のゼロの確率ではありません。真のp値は.00004や.0000001かもしれませんが、APA形式では単に *p* < .001と要求されます。

### ルール3：小数点の前に先頭ゼロを付けない

p値は0と1の間に制限され、1.0以上になることは決してないため、APA形式では先頭ゼロを省略します。

- 正しい：*p* = .042
- 不正確：*p* = 0.042

このルールは、絶対値で1.0を超えることのないすべての統計量に適用されます。これにはp値、相関係数（*r*）、割合、クロンバックのアルファが含まれます。コーエンの *d*、平均値、標準偏差など、1.0を超えることがある統計量には適用されません。

### ルール4：*p* の文字をイタリック体にする

APA形式では、母集団パラメータに由来するまたは慣例的な略語として機能する統計記号はイタリック体にされます。p値の *p* は常にイタリック体です。

- 正しい：*p* = .017
- 不正確：p = .017（イタリック体でない）

イタリック体にすべき他の統計記号には、*t*、*F*、*r*、*R*²、*d*、*M*、*SD*、*N*、*n*、*df*、および統計量として使用されるギリシャ文字（偏イータ二乗など、*η*p²と表記）があります。

### ルール5：演算子の前後にスペースを入れる

等号または不等号の前後にスペースを置いてください。

- 正しい：*p* = .032
- 正しい：*p* < .001
- 不正確：*p*=.032（スペースなし）
- 不正確：*p*<.001（スペースなし）

これはAPA形式のすべての数学的演算子（=、<、>、マイナス記号を含む）に適用されます。

### まとめ：正しいp値書式と不正確な書式

| 正しい | 不正確 | 間違っている理由 |
|--------|--------|----------------|
| *p* = .032 | p = .032 | *p* はイタリック体にすべき |
| *p* = .032 | *p* = 0.032 | 1を超えない値には先頭ゼロなし |
| *p* = .032 | *p* = .03 | 小数点以下3桁に丸める |
| *p* < .001 | *p* = .000 | 確率は決して正確にゼロではない |
| *p* < .001 | *p* = .0001 | 非常に小さい値には *p* < .001を使用 |
| *p* = .032 | *p*=.032 | = の前後にスペースが必要 |
| *p* = .871 | *p* = n.s. | 非有意でも正確な値を報告 |

## 一般的な検定ごとのp値報告

各統計検定には独自のAPA報告テンプレートがあります。以下は、最も一般的な7つの検定の具体的な数値例です。APA第7版が4つすべてを要求するため、すべての例に検定統計量、自由度、正確なp値、効果量が含まれています。

### 対応のないt検定

**テンプレート：** *t*(df) = X.XX, *p* = .XXX, *d* = X.XX

**有意な例：**

> Students in the intervention group (*M* = 82.4, *SD* = 11.3) scored significantly higher than the control group (*M* = 74.1, *SD* = 12.7), *t*(78) = 3.12, *p* = .003, *d* = 0.70.

**非有意な例：**

> There was no statistically significant difference in reaction time between the caffeine group (*M* = 312 ms, *SD* = 45) and the placebo group (*M* = 319 ms, *SD* = 51), *t*(62) = 0.61, *p* = .544, *d* = 0.15.

コーエンの *d* は1.0を超えることがあるため、先頭ゼロを使用することに注意してください。

### 対応のあるt検定

**テンプレート：** *t*(df) = X.XX, *p* = .XXX, *d* = X.XX

**例：**

> Anxiety scores decreased significantly from pre-test (*M* = 38.2, *SD* = 8.4) to post-test (*M* = 31.7, *SD* = 7.9), *t*(44) = 4.58, *p* < .001, *d* = 0.68.

対応のあるt検定の自由度は *n* - 1で、*n* は対の数です。

### 一元配置分散分析

**テンプレート：** *F*(df1, df2) = X.XX, *p* = .XXX, *η*p² = .XX

**例：**

> There was a statistically significant difference in job satisfaction across the three departments, *F*(2, 87) = 5.94, *p* = .004, *η*p² = .12.

### カイ二乗独立性の検定

**テンプレート：** *χ*²(df, *N* = XXX) = X.XX, *p* = .XXX, *V* = .XX

**例：**

> There was a significant association between gender and preferred learning style, *χ*²(2, *N* = 180) = 8.74, *p* = .013, *V* = .22.

### ピアソン相関

**テンプレート：** *r*(df) = .XX, *p* = .XXX

**例：**

> There was a strong positive correlation between study hours and exam performance, *r*(88) = .62, *p* < .001.

相関の自由度は *N* - 2です。*r* は -1から1の範囲のため先頭ゼロは付けません。分散説明量を伝えたい場合は *r*² を報告してください：*r*² = .38は、学習時間が試験成績の分散の38%を説明していることを意味します。

### 単回帰・重回帰

**モデル全体のテンプレート：** *F*(df1, df2) = X.XX, *p* = .XXX, *R*² = .XX

**個々の予測変数のテンプレート：** *β* = X.XX, *t*(df) = X.XX, *p* = .XXX

**例：**

> The regression model was statistically significant, *F*(3, 96) = 14.82, *p* < .001, *R*² = .32. Hours of practice (*β* = .41, *t*(96) = 4.23, *p* < .001) and years of experience (*β* = .28, *t*(96) = 2.87, *p* = .005) were significant predictors, while age was not (*β* = .09, *t*(96) = 0.91, *p* = .365).

### Mann-Whitney U検定

**テンプレート：** *U* = XXX, *z* = X.XX, *p* = .XXX, *r* = .XX

**例：**

> Satisfaction ratings were significantly higher in the treatment group (*Mdn* = 4.5) than the control group (*Mdn* = 3.0), *U* = 245, *z* = -2.89, *p* = .004, *r* = .36.

Mann-Whitney Uの記述統計量には、平均値と標準偏差ではなく中央値と四分位範囲を報告してください。

## 有意な結果と非有意な結果

### 有意な結果の報告（*p* < .05）

p値が事前に設定したアルファ水準（通常.05）を下回った場合、結果を「統計的に有意」と報告してください。正確なp値を使用し、効果量と対にしてください。

> The difference was statistically significant, *t*(58) = 2.41, *p* = .019, *d* = 0.63.

発見を誇張する表現は避けてください。「仮説が証明された」や「治療が確実に効く」とは書かないでください。

### 非有意な結果の報告（*p* > .05）

p値がアルファを超えた場合、結果を「統計的に有意ではなかった」と報告してください。それでも正確なp値と効果量を報告してください。

> The difference was not statistically significant, *t*(58) = 1.14, *p* = .259, *d* = 0.30.

**重要な用語：** 非有意な結果を「insignificant（取るに足らない）」と表現しないでください。日常英語では「insignificant」は「重要でない」を意味しますが、これは統計的非有意が意味するところではありません。

また以下も避けてください：
- 「結果が帰無仮説を証明した」（帰無を証明することはできない）
- 「効果はなかった」（証拠の不在は不在の証拠ではない）
- 「群は同一であった」（非有意は同等性を実証しない）

代わりに、「統計的に有意な差はなかった」または「結果は統計的有意に達しなかった」と書いてください。

### 辺縁的に有意な結果（*p* = .05 ～ .10）

p値が.05と.10の間にある結果は、議論のある領域を占めます。

**推奨されるアプローチ：** 正確なp値、効果量、信頼区間を報告してください。データ自体に語らせ、主観的なラベルを適用しないでください。

> The difference did not reach statistical significance, *t*(28) = 1.89, *p* = .069, *d* = 0.71, 95% CI [-0.05, 1.46].

効果量（*d* = 0.71）と信頼区間（ほぼゼロを除外している）が、p値だけよりもはるかに豊かな物語を伝えていることに注目してください。

### 常に効果量を含める

APA第7版（セクション6.36）は明確です：「読者が研究の知見の大きさまたは重要性を理解するためには、何らかの効果量の指標を含めることがほぼ常に必要である。」

結果が有意であるかどうかに関わらず、すべての推測統計検定に対して適切な効果量を報告してください：

| 検定 | 効果量 | 小 | 中 | 大 |
|------|--------|---|---|---|
| t検定 | Cohen's *d* | 0.20 | 0.50 | 0.80 |
| 分散分析 | 偏 *η*² | .01 | .06 | .14 |
| カイ二乗 | Cramérの *V* | .10 | .30 | .50 |
| 相関 | *r* | .10 | .30 | .50 |
| 回帰 | *R*² | .02 | .13 | .26 |
| Mann-Whitney U | *r* | .10 | .30 | .50 |

## 片側検定と両側検定

### それぞれの使用場面

**両側検定** は、効果がいずれの方向にも存在するかどうかを評価します。事実上すべての研究文脈でデフォルトです。

**片側検定** は、効果が特定の事前指定された方向に存在するかどうかを評価します。より高い統計的検出力を持ちますが、予測された方向の効果のみを検出できます。

### APA形式での報告方法

片側検定を使用する場合、これを明示的に述べ、方向性のある予測を正当化しなければなりません。

**両側（デフォルト、特別な記載は不要）：**

> *t*(58) = 2.41, *p* = .019, *d* = 0.63.

**片側（宣言が必要）：**

> Based on previous findings that the intervention increases reading speed, we used a one-tailed test. The intervention group read significantly faster, *t*(58) = 1.82, *p* = .037 (one-tailed), *d* = 0.47.

## よくある間違いと回避方法

### 1. *p* = .000と書く

最も一般的なp値のエラーです。SPSSやExcelなどのプログラムは、p値が小数点以下3桁でゼロに丸められるときに.000と表示します。*p* < .001と書いてください。

### 2. 先頭ゼロを付ける

*p* = .042の代わりに *p* = 0.042と書くことはAPA形式に違反します。

### 3. 正確なp値を報告しない

正確な値なしに *p* < .05と書くことは、以前のAPA版では許容されていましたが、もはや十分ではありません。APA第7版は正確な値（例：*p* = .032）を要求します。唯一の例外は *p* < .001の場合です。

### 4. 仮説が「証明された」と言う

統計検定は仮説を証明しません。「結果が仮説を支持した」または「帰無仮説が棄却された」と書いてください。

### 5. 有意性ラベルのみを報告する

検定統計量、自由度、正確なp値、効果量を提供せずに「結果は有意であった」と書くことは、読者が証拠を評価することを不可能にします。

### 6. *p* をイタリック体にしない

APA原稿では、*p* は常にイタリック体です。これは原稿テキスト、表、図に適用されます。

### 7. 統計的有意性と実践的重要性を混同する

*p* = .001で *d* = 0.08の結果は統計的に有意ですが実践的に些細です。逆に、*p* = .08で *d* = 0.75の結果は統計的に有意ではありませんが、検出力不足だった可能性のある実質的な効果を示唆しています。常にp値を効果量と併せて解釈してください。

### 8. 効果量を報告しない

APA第7版は、すべての推測統計検定に対して効果量を義務づけています。

### 9. 正当化なしに「辺縁的に有意」を使用する

結果を「辺縁的に有意」と記述する場合、査読者からの異議に備えてください。

### 10. 「有意に接近している」と書く

この表現は「辺縁的に有意」よりもさらに問題があります。p値は何かに「接近」しません。値はそのままです。正確な数値を報告し、読者に判断を委ねてください。

## p値の閾値：アルファ = .05、.01、.001

### 各閾値の意味

アルファ水準（*α* と表記）は、データ分析*前*に設定する統計的有意性の基準です。

- ***α* = .05** は、第I種の誤り（真の帰無仮説を誤って棄却する）の5%の確率を受け入れることを意味します。大部分の社会・行動科学での標準です。
- ***α* = .01** は1%の確率を受け入れます。より保守的な証拠が必要な分野や状況で使用されます。
- ***α* = .001** は0.1%の確率を受け入れます。ゲノムワイド関連解析や素粒子物理学のような非常に高い証拠基準が適用される文脈で使用されます。

### より厳しい閾値を使用すべき場合

**多重比較。** 同一データセットで多数の検定を実行すると、少なくとも1つの偽陽性の確率が増加します。Bonferroni修正はアルファを比較回数で除します。

**ハイステークスな決定。** 薬剤承認のための臨床試験、安全性が重要なエンジニアリングの決定、政策的影響の大きい研究では、*α* = .01以上の厳しさが正当化される場合があります。

### 閾値の報告方法

方法セクションでアルファ水準を述べ、一貫して適用してください。

> Statistical significance was evaluated at *α* = .05 for all analyses. For post-hoc pairwise comparisons, a Bonferroni-adjusted alpha of .017 was used (.05 / 3 comparisons).

## 提出前のチェックリスト

提出する前に、原稿のすべてのp値について以下のチェックリストを確認してください：

- *p* がイタリック体か
- = または < の前後にスペースがあるか
- 値が小数点以下3桁で報告されているか
- *p* = .000ではなく *p* < .001と書いているか
- 小数点の前に先頭ゼロがないか
- 正確なp値を報告しているか（*p* < .05だけではなく）
- p値とともに効果量が報告されているか
- 記述統計量（平均値、SD、または中央値）が含まれているか
- 検定統計量とともに自由度が含まれているか
- 「証明された」「insignificant」「有意に接近している」などの表現を避けているか

すべてにチェックが入れば、p値の報告はAPA第7版の基準を満たしています。

## p値を自動フォーマットするツール

p値を手作業でフォーマットするのは面倒でエラーが発生しやすい作業です。以下のツールが役立ちます：

- **StatMate**（[statmate.org](https://statmate.org)）は、t検定、分散分析、カイ二乗検定、相関、回帰を含む20の統計検定に対してAPA形式の結果文を自動生成します。データを入力すると、正しいp値の書式、効果量、信頼区間を含む出版可能な文が生成されます。結果を原稿に直接コピーできます。
- **R** の `papaja` パッケージの `apa_print()` 関数はフォーマット済みの統計量を出力します。
- **JASP** と **jamovi** はデフォルト出力にAPA形式の表を含みます。
- **SPSS** はAPA形式でp値をフォーマットしません。「Sig. = .000」を *p* < .001に手動で変換し、先頭ゼロを削除する必要があります。

## まとめ

APA第7版形式でp値を報告するには5つのことが必要です：小数点以下3桁の正確な値、非常に小さい値に対する *p* < .001、先頭ゼロなし、イタリック体の *p*、演算子の前後のスペース。書式を超えて、APAはp値が常に効果量とともに、理想的には信頼区間を伴うべきであることを強調しています。

最も重要な原則はこれです：p値は証拠の一片であり、物語全体ではありません。適切に報告された結果には、記述統計量、自由度付きの検定統計量、正確なp値、効果量、信頼区間が含まれます。これらの要素を合わせることで、読者があなたの知見を評価するために必要なすべてが提供されます。
