## Kruskal-Wallis検定を使用する場面

Kruskal-Wallis H検定は、一元配置分散分析のノンパラメトリック代替法です。3つ以上の独立群間で、連続変数または順序変数の分布を比較します。以下の条件のうち少なくとも1つが当てはまる場合に、Kruskal-Wallis検定を選択すべきです。

- 1つ以上の群で従属変数が正規分布していない（Shapiro-Wilk *p* < .05）
- 分散の等質性の仮定が破られている（Leveneの検定 *p* < .05）
- 群ごとのサンプルサイズが小さい（一般的に*n* < 20）
- 正当化も除外もできない極端な外れ値がデータに含まれている
- 変数が順序尺度で測定されている（例：Likert尺度の評定）

Kruskal-Wallis検定の結果を報告する際、APA第7版ではノンパラメトリック検定を選択した理由を説明することが求められます。違反した仮定についての一文で十分です。

## APAの報告テンプレート

Kruskal-Wallis検定の結果を報告する標準的なAPA形式は以下の通りです。

> *H*(df) = X.XX, *p* = .XXX, *e*^2^ = .XX

各要素の意味は以下の通りです。

- ***H***：Kruskal-Wallis検定統計量（カイ二乗分布に従う）
- **df**：自由度。群の数から1を引いた値（*k* - 1）
- ***p***：小数第3位までの正確な*p*値（.001未満の場合は*p* < .001）
- **効果量**：イプシロン二乗（*e*^2^）またはイータ二乗（*n*^2^~H~）

効果量は必ず含めてください。有意な*p*値は何かが異なることを示しますが、効果量はどの程度異なるかを示します。

## ステップバイステップの報告方法

### ステップ1：検定名を述べ、選択を正当化する

まず検定名を示し、ノンパラメトリック代替法を選択した理由を簡潔に説明します。

> 3つの治療群間で痛みの評定を比較するため、Kruskal-Wallis H検定を実施した。2群で痛みの評定が正規性の仮定に違反していたため（Shapiro-Wilk *p* < .05）、ノンパラメトリック検定を選択した。

### ステップ2：中央値とIQRによる記述統計を報告する

ノンパラメトリック検定では、平均値と標準偏差ではなく、**中央値**（*Mdn*）と四分位範囲（IQR）を報告します。平均値は対称分布を仮定しますが、それはまさに違反を認めた仮定そのものです。

**シナリオの例：** 3つの治療群間で痛みの評定（0-10スケール）を比較する。

| 群 | *n* | *Mdn* | IQR |
|----|-----|-------|-----|
| プラセボ | 30 | 7.00 | 5.00-8.00 |
| 薬剤A | 30 | 5.00 | 3.00-6.50 |
| 薬剤B | 30 | 3.00 | 2.00-5.00 |

> 痛みの評定の中央値は、プラセボ群で最も高く（*Mdn* = 7.00, IQR = 5.00-8.00）、次いで薬剤A（*Mdn* = 5.00, IQR = 3.00-6.50）、薬剤B（*Mdn* = 3.00, IQR = 2.00-5.00）の順であった。

### ステップ3：H統計量をdfおよびpとともに報告する

> Kruskal-Wallis検定により、3つの治療群間で痛みの評定に統計的に有意な差が認められた（*H*(2) = 24.37, *p* < .001）。

書式の重要なルール：

- *H*統計量をイタリック体にする
- 自由度を*H*の直後にスペースなしで括弧内に記載する
- 正確な*p*値を小数第3位まで報告する
- *p* < .001を下限として使用する

### ステップ4：効果量を報告する

> 効果は大きかった（*e*^2^ = .27）。

効果量を含めることで、分析が単なる「有意かどうか」の二値判断から、実践的重要性についての意味のある記述へと変わります。

## Kruskal-Wallis検定と一元配置分散分析：選択の基準

Kruskal-Wallis検定と一元配置分散分析の選択は、データの特性と各検定が要求する仮定に依存します。これらのトレードオフを理解することで、査読者に対して分析方法の選択を正当化できます。

### 仮定の比較

| 基準 | 一元配置分散分析 | Kruskal-Wallis検定 |
|------|-----------------|-------------------|
| 分布 | 正規（群ごと） | 任意の分布 |
| 尺度 | 連続（間隔/比率） | 順序または連続 |
| 中心傾向 | 平均値を比較 | 平均順位を比較 |
| サンプルサイズ | 群あたり*n* > 30で頑健 | 任意のサンプルサイズ |
| 外れ値 | 外れ値に敏感 | 外れ値に頑健 |
| 等分散性 | 必要（Leveneの検定） | 不要 |
| 検定力 | 仮定充足時により高い | 分散分析よりやや低い |
| 事後検定 | Tukey HSD, Bonferroni | Dunnの検定 |
| 効果量 | イータ二乗（*n*^2^） | イプシロン二乗（*e*^2^） |

### 検定力のトレードオフ

すべてのANOVAの仮定が満たされている場合、Kruskal-Wallis検定はパラメトリックANOVAの検定力の約95%を持ちます。つまり、同じサンプルサイズでは、Kruskal-Wallis検定は真の差を検出する可能性がわずかに低くなります。しかし、仮定が違反されている場合——特に歪んだ分布や外れ値がある場合——パラメトリック検定の検定力は仮定違反の下で低下するため、Kruskal-Wallis検定の方が実際にはANOVAよりも*強力*になり得ます。

実践的な含意：Kruskal-Wallis検定を「安全な」選択肢として自動的に選ばないでください。データが本当にANOVAの仮定を満たしているなら、ノンパラメトリック代替法に切り替えることで不必要に統計的検定力を犠牲にすることになります。まず仮定の検証を行い、違反が検出された場合にのみ切り替えてください。

### APAでの正当化テンプレート

方法のセクションで選択を正当化する際は、以下のテンプレートのいずれかを使用してください。

> **Kruskal-Wallis検定を選択する場合：** 3群中2群で従属変数が正規分布していなかったため（Shapiro-Wilk *p* = .012および*p* = .003）、Kruskal-Wallis H検定を使用した。

> **ANOVAを選択する場合：** 正規性（すべての群でShapiro-Wilk *p* > .05）および分散の等質性（Leveneの検定 *p* = .318）の仮定が満たされたため、一元配置分散分析を実施した。

## Kruskal-Wallis検定後の事後検定

有意なKruskal-Wallis検定の結果は、少なくとも1つの群が少なくとも他の1群と異なることを示しますが、*どの*群が異なるかは示しません。全体検定が有意な場合は、対の事後比較を実施する必要があります。

### Bonferroni補正付きDunnの検定

Dunnの検定は、Kruskal-Wallis検定の標準的な事後手続きです。順位和を使用してすべての可能な群の対を比較し、多重比較の補正を行います。Dunnの検定は同じ順位付けの枠組みを使用するため、個別のMann-Whitney U検定を実施するよりも適切です。

各対比較の検定統計量は*z*値であり、平均順位の差をその差の標準誤差で割って計算されます。Bonferroni補正は、有意水準（アルファ）を比較の数で割り、ファミリーワイズの第一種の過誤率を制御します。

**Dunnの検定のAPA報告形式：**

> Bonferroni補正を用いたDunnの事後対比較の結果、薬剤Bはプラセボ群（*z* = -4.82, *p* < .001）および薬剤A（*z* = -2.67, *p* = .023）よりも有意に低い痛みの評定を示した。薬剤Aとプラセボ群の差も有意であった（*z* = -2.15, *p* = .047）。

### 対のMann-Whitney U検定

代替的な事後検定のアプローチとして、各群の対に対して個別のMann-Whitney U検定を実施することができます。このアプローチは、各比較でデータを再順位付けするため、Dunnの検定よりも統計的に非効率です。しかし、一部の査読者は、馴染みのある*U*統計量と各対の順位双列相関効果量を提供するため、この方法を好みます。

対のMann-Whitney U検定を使用する場合、多重比較補正を**必ず**適用しなければなりません。補正なしでは、ファミリーワイズの過誤率が急速に増大します。3群の場合、3回の比較でファミリーワイズのアルファは.05ではなく約.14になります。

**対のMann-Whitney UのAPA報告形式：**

> 事後比較として、Bonferroni補正を用いた対のMann-Whitney U検定を実施した。薬剤Bはプラセボ群（*U* = 142.00, *z* = -4.78, *p* < .001, *r* = -.62）および薬剤A（*U* = 274.50, *z* = -2.71, *p* = .020, *r* = -.35）よりも有意に低い痛みの評定を示した。薬剤Aとプラセボ群の差も有意であった（*U* = 305.00, *z* = -2.19, *p* = .043, *r* = -.28）。

### 補正方法の選択

| 補正方法 | 計算式 | 保守性 | 最適な使用場面 |
|----------|--------|--------|---------------|
| Bonferroni | *a* / *m* | 最も保守的 | 比較数が少ない場合、高リスク研究 |
| Holm | ステップダウンBonferroni | やや保守的でない | ほとんどの研究のデフォルト推奨 |
| Benjamini-Hochberg | FDRに基づく | 最も保守的でない | 探索的分析、比較数が多い場合 |

有意な比較と有意でない比較がある場合は、両方を報告します。

> Bonferroni補正を用いたDunnの事後検定の結果、薬剤Bはプラセボ群と有意に異なったが（*z* = -4.82, *p* < .001）、薬剤Aとは有意差がなかった（*z* = -1.94, *p* = .157）。

## Kruskal-Wallis検定の効果量：イプシロン二乗とイータ二乗

効果量は、Kruskal-Wallis検定結果の実践的有意性を解釈するために不可欠です。APA第7版はすべての統計検定について効果量の報告を明示的に推奨しており、Kruskal-Wallis検定も例外ではありません。

### イプシロン二乗（*e*^2^）

イプシロン二乗は、Kruskal-Wallis検定で最も一般的に報告される効果量です。群分け変数によって説明される順位の分散の割合を推定します。計算式は以下の通りです。

> *e*^2^ = *H* / ((*N*^2^ - 1) / (*N* + 1))

ここで、*N*は総サンプルサイズ、*H*は検定統計量です。これを簡略化すると：

> *e*^2^ = *H* / (*N* - 1)

| *e*^2^ | 解釈 |
|--------|------|
| .01 | 小さい効果 |
| .06 | 中程度の効果 |
| .14 | 大きい効果 |

**計算例：** *H* = 24.37、*N* = 90の場合：

> *e*^2^ = 24.37 / (90 - 1) = 24.37 / 89 = .27

これは大きな効果であり、順位の分散の27%が群の所属によって説明されることを示します。

**APA形式：**

> 効果は大きく（*e*^2^ = .27）、痛みの評定順位の変動性の約27%が治療群によって説明された。

### Hに基づくイータ二乗（*n*^2^~H~）

代替指標として、群の数を補正するHに基づくイータ二乗があります。

> *n*^2^~H~ = (*H* - *k* + 1) / (*N* - *k*)

ここで、*k*は群の数、*N*は総サンプルサイズです。

| *n*^2^~H~ | 解釈 |
|-----------|------|
| .01 | 小さい効果 |
| .06 | 中程度の効果 |
| .14 | 大きい効果 |

**計算例：** *H* = 24.37、*k* = 3、*N* = 90の場合：

> *n*^2^~H~ = (24.37 - 3 + 1) / (90 - 3) = 22.37 / 87 = .26

両方の指標は同じベンチマークを共有しています。イプシロン二乗は社会科学で広く使用され、イータ二乗は保健科学の文献でより頻繁に見られます。どちらか一方を選び、論文全体で一貫して使用してください。

### 対の効果量

事後比較では、各対について順位双列相関（*r*）を報告します。これは2つの特定の群がどの程度異なるかの標準化された指標です。

> *r* = *Z* / sqrt(*N*~pair~)

ここで、*Z*は対比較の標準化された検定統計量、*N*~pair~は比較される2群の合計サンプルサイズです。ベンチマークはCohenの慣例に従います：.10小さい効果、.30中程度の効果、.50大きい効果。

## 複数の従属変数でのKruskal-Wallis検定の報告

研究で同じ群を複数のアウトカム変数について比較する場合、結果を明確に整理することが重要になります。各Kruskal-Wallis検定を本文中で個別に報告すると、密集して読みにくいパラグラフになります。代わりに、構造化されたAPA表を使用してください。

### 複数アウトカムのAPA表形式

**Table 1**

*治療群間のKruskal-Wallis H検定結果*

| 従属変数 | *H*(2) | *p* | *e*^2^ | 有意な対 |
|----------|--------|-----|--------|----------|
| 痛みの評定 | 24.37 | < .001 | .27 | B < A < P |
| 不安得点 | 18.92 | < .001 | .21 | B < P; A = P |
| 睡眠の質 | 8.45 | .015 | .10 | B > P |
| 疲労度 | 3.12 | .210 | .04 | — |

*注.* N = 90（群あたり30名）。P = プラセボ、A = 薬剤A、B = 薬剤B。有意な対はBonferroni補正付きDunnの検定により決定（*p* < .05）。ダッシュは有意な対差がないことを示す。

### 結果パラグラフの記述

複数のKruskal-Wallis検定を報告する場合、表を参照し主要なパターンを強調する簡潔な記述的要約を提供してください。

> 3つの治療群を4つのアウトカム指標について比較するため、Kruskal-Wallis H検定を実施した（Table 1参照）。痛みの評定（*H*(2) = 24.37, *p* < .001, *e*^2^ = .27）、不安得点（*H*(2) = 18.92, *p* < .001, *e*^2^ = .21）、睡眠の質（*H*(2) = 8.45, *p* = .015, *e*^2^ = .10）に有意な群間差が認められた。疲労度には有意差がなかった（*H*(2) = 3.12, *p* = .210, *e*^2^ = .04）。Bonferroni補正付きDunnの事後対比較の詳細はTable 1に示した。

### 多重検定の処理

同じサンプルから異なる従属変数に対して複数のKruskal-Wallis検定を実施する場合、多重検定の補正が必要かどうかを検討すべきです。従属変数が概念的に関連しており、仮説群を検定している場合は、全体の*p*値にBonferroniまたはHolm補正を適用してください。従属変数が概念的に異なるアウトカムを表している場合、補正は不要であるというのが多くの統計学者の見解です。いずれの場合も、その根拠を述べてください。

## APAでの完全な報告例

以下は、上述の痛みの評定シナリオのすべての要素を組み合わせた完全な結果パラグラフです。

> 3つの治療条件間での痛みの評定（0-10スケール）の差を検討するため、Kruskal-Wallis H検定を実施した。対象はプラセボ（*n* = 30）、薬剤A（*n* = 30）、薬剤B（*n* = 30）であった。プラセボ群と薬剤B群において痛みの評定が正規性の仮定に違反していたため（Shapiro-Wilk *p* = .012および*p* = .003）、ノンパラメトリック検定を選択した。痛みの評定の中央値は、プラセボ群7.00（IQR = 5.00-8.00）、薬剤A 5.00（IQR = 3.00-6.50）、薬剤B 3.00（IQR = 2.00-5.00）であった。Kruskal-Wallis検定は群間の痛みの評定に統計的に有意な差を示した（*H*(2) = 24.37, *p* < .001, *e*^2^ = .27）。Bonferroni補正を用いたDunnの事後対比較の結果、薬剤Bとプラセボ（*z* = -4.82, *p* < .001）、薬剤Bと薬剤A（*z* = -2.67, *p* = .023）、薬剤Aとプラセボ（*z* = -2.15, *p* = .047）の間に有意差が認められた。

このパラグラフには、必要なすべての要素が含まれています。検定の正当化、中央値とIQRによる記述統計、自由度と*p*値を伴う全体*H*統計量、効果量、事後対比較です。

## Kruskal-Wallis検定のAPA報告でよくある間違い

### 1. 中央値ではなく平均値を報告する

これは最も頻繁に見られる誤りです。分布が非正規であるためにノンパラメトリック検定を選択したのであれば、平均値と標準偏差を報告することは検定選択の根拠と矛盾します。常に中央値と四分位範囲を報告してください。

**誤り：**
> 群A（*M* = 5.23, *SD* = 2.14）、群B（*M* = 3.87, *SD* = 1.92）

**正しい：**
> 群A（*Mdn* = 5.00, IQR = 3.50-7.00）、群B（*Mdn* = 4.00, IQR = 2.00-5.50）

全体像を提供するために平均値と中央値の両方を報告する研究者もおり、中央値が本文で議論される主要な記述統計である限り許容されます。

### 2. 効果量を省略する

*p*値だけでは不十分です。APA第7版では、すべての統計検定について効果量の報告を明示的に推奨しています。Kruskal-Wallis検定では、イプシロン二乗またはイータ二乗を含めてください。効果量がなければ、観察された差が些細なものか実践的に意味のあるものかを読者が判断できません。

### 3. 有意な全体検定後に事後検定を省略する

3群以上で有意な*H*統計量が得られた場合は、対のフォローアップが必要です。事後比較なしでは、読者はどの特定の群が異なるかを判断できません。常にDunnの検定（または同等の手続き）を実施し、調整済み*p*値を報告してください。唯一の例外は群が2つしかない場合で、その場合は全体検定自体が対比較です。

### 4. 効果量を誤って計算する

一般的な計算エラーは、イプシロン二乗に誤った公式を使用することです。一部の研究者は誤ってANOVAのイータ二乗の公式（SS~between~ / SS~total~）を使用しますが、この公式にはKruskal-Wallis検定の出力に含まれない生データの平方和が必要です。正しい公式は*H*統計量と総サンプルサイズのみを使用します：*e*^2^ = *H* / (*N* - 1)。

### 5. *n* < 20のときに正確検定を報告しない

サンプルサイズが非常に小さい場合（総*N* < 20または群*n* < 5）、標準的なKruskal-Wallis検定で使用されるカイ二乗近似は信頼性が低くなります。このような場合は、正確な並べ替え検定を使用し、明示的に報告すべきです。

> 群サイズが小さいため（*n*~1~ = 4, *n*~2~ = 5, *n*~3~ = 4）、正確Kruskal-Wallis検定を実施した（*H*(2) = 7.65, exact *p* = .018）。

ほとんどの統計ソフトウェア（SPSS、R、StatMate）は正確検定を計算できます。小さなサンプルで漸近（カイ二乗）近似を使用した場合、査読者が方法論上の懸念として指摘する可能性があります。

### 6. Kruskal-Wallis検定とFriedman検定を混同する

Kruskal-Wallis検定とFriedman検定はどちらも3つ以上の群を比較するノンパラメトリック代替法ですが、異なるデザインに対応しています。Kruskal-Wallis検定は**独立群**（各群に異なる参加者）用であり、Friedman検定は**反復測定**（同じ参加者がすべての条件で測定される）用です。誤った検定を使用することは、報告の問題ではなく、根本的なデザインの誤りです。

| 特徴 | Kruskal-Wallis検定 | Friedman検定 |
|------|-------------------|-------------|
| デザイン | 独立群 | 反復測定 |
| パラメトリック等価物 | 一元配置分散分析 | 反復測定分散分析 |
| 順位付け | すべての観測値を一緒に順位付け | 各参加者内で順位付け |
| 事後検定 | Dunnの検定 | Nemenyiの検定またはConoverの検定 |
| 効果量 | イプシロン二乗 | Kendallの*W* |

### 7. ノンパラメトリック検定を使用した理由を説明しない

査読者はなぜより検出力の高い一元配置分散分析を使用しなかったかの説明を期待します。違反した特定の仮定に言及する一文で十分です。

**弱い：**
> Kruskal-Wallis検定を使用した。

**強い：**
> 3群中2群で痛みの評定が正規分布していなかったため（Shapiro-Wilk *p* < .05）、Kruskal-Wallis検定を使用した。

### 8. 補正方法を明記し忘れる

事後対比較を報告する際は、常にどの多重比較補正を適用したか（例：Bonferroni、Holm、Benjamini-Hochberg）を述べてください。この情報がなければ、読者は分析を評価したり再現したりすることができません。

## よくある質問

### Kruskal-Wallis検定を2群だけに使用できますか？

技術的には、2群に対するKruskal-Wallis検定はMann-Whitney U検定と同等の結果を生みます。しかし、2群の比較にはMann-Whitney U検定が標準的な選択です。これは、より有益で査読者に馴染みのある追加の統計量（*U*統計量、順位双列相関）を提供するためです。3群以上にはKruskal-Wallis検定を、ちょうど2群にはMann-Whitney U検定を使用してください。

### Kruskal-Wallis検定の結果とともにどの記述統計を報告すべきですか？

主要な記述統計として中央値（*Mdn*）と四分位範囲（IQR）を報告してください。群の順序を明確にするのに役立つ場合は、平均順位も報告できます。平均値と標準偏差を唯一の記述統計として報告することは避けてください。これは対称分布を仮定するものであり、ノンパラメトリック検定を選択した根拠と矛盾します。文脈のために平均値を報告したい場合は、中央値とともに提示し、中央値が適切な要約であることを明示してください。

### Kruskal-Wallis検定は何群まで比較できますか？

群の数に理論上の上限はありません。しかし、群の数が増えると、対の事後比較の数が急速に増大します（*k*群に対して*k*(*k* - 1)/2比較）。5群なら10の対比較、7群なら21になります。比較数が多いほど補正が厳しくなり、比較あたりの検定力が低下します。5-6群を超える場合は、一部の群を統合できるか、異なる分析アプローチ（例：計画対比）がより適切かを検討してください。

### Kruskal-WallisのH統計量とカイ二乗の関係は何ですか？

*H*統計量は*k* - 1の自由度を持つ近似カイ二乗分布に従います。ここで*k*は群の数です。この近似は各群に少なくとも5つの観測値がある場合に適切です。より小さなサンプルでは、正確な並べ替え分布を代わりに使用すべきです。*p*値がカイ二乗分布表から導出されるのは、このカイ二乗近似によるものです。

### 平均順位と中央値のどちらを報告すべきですか？

どちらも有効ですが、中央値は元の測定尺度で直接解釈可能であるため、APA形式の報告ではより一般的に期待されます。平均順位は、各群の観測値が全体の順位付けで占める平均位置を示しており、これがKruskal-Wallis検定が実際に分析しているものです。群が類似した分布を持ちながら位置だけが異なる場合、平均順位と中央値は同じ結論を指します。分布が位置だけでなく形状も異なる場合は、平均順位と中央値が乖離する可能性があり、両方を報告すべきです。

### タイ値（同順位）はどのように処理しますか？

タイ値は、占めるはずだった順位の平均を受け取ります。例えば、2つの観測値が順位5と6でタイの場合、両方とも順位5.5を受け取ります。ほとんどのソフトウェアは*H*統計量にタイ補正を自動的に適用します。補正は*H*値をわずかに増加させ、検定をやや保守的にします。タイが多い場合（順序データや丸められた測定で一般的）、ソフトウェアがタイ補正を適用したことを常に確認し、報告書にその旨を記載してください。

### Likert尺度のような順序データにKruskal-Wallis検定を使用できますか？

はい。Kruskal-Wallis検定は順序データと非正規の連続データのために特別に設計されています。個々のLikert項目（例：単一の1-5項目）については、データが順序型であるためKruskal-Wallis検定が適切です。複合Likert尺度（複数項目の平均値または合計値）については、多くの研究者が複合点を連続変数として扱いANOVAを使用しますが、複合点が正規分布していない場合はKruskal-Wallis検定も有効な選択です。

### Kruskal-Wallis検定が有意でない場合はどうすればよいですか？

効果量を含めて結果を完全に報告してください：*H*(df) = X.XX, *p* = .XXX, *e*^2^ = .XX。統計的に有意な差が認められなかったことを述べてください。全体検定が群間差を示さなかったため、事後比較は実施しないでください。統計的有意性がなくても読者が観察された差の大きさを評価できるよう、記述統計（中央値、IQR）を報告してください。小さな効果量と十分な検定力を持つ有意でない結果は、群が類似しているという有用な証拠を提供します。
