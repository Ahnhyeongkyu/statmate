## Mann-Whitney U検定が必要な場合

Mann-Whitney U検定（Wilcoxon順位和検定とも呼ばれる）は、対応のないt検定のノンパラメトリック代替法です。2つの独立グループを順序変数または連続変数で比較する際に使用し、以下の条件の1つ以上に該当する場合に適しています：

- 従属変数が一方または両方のグループで正規分布に従わない
- サンプルサイズが小さい（一般的にグループあたり *n* < 30）
- データに顕著な外れ値がある
- 変数が順序尺度で測定されている

最も広く使用されるノンパラメトリック検定の一つであるにもかかわらず、多くの研究者がMann-Whitney Uの結果をAPA形式で正しく報告することに苦労しています。このガイドでは、明確なテンプレートと実際の例を提供します。

## APA報告の必須要素

APA第7版でMann-Whitney Uの結果を報告する際に含めるべき要素：

- **U統計量**：Mann-Whitney *U* 値
- **サンプルサイズ**：各グループの *n*
- **z得点**：標準化された検定統計量（大標本の場合）
- **正確な *p* 値**：小数点第3位まで
- **効果量**：順位双列相関（*r*）または *r* = *z* / &radic;*N*
- **中央値と四分位範囲**：各グループごと

## ステップ1：記述統計の報告

ノンパラメトリック検定では、平均と標準偏差の代わりに**中央値**（*Mdn*）と四分位範囲（IQR）を報告します。

**シナリオ例：** 治療群（*n* = 25）と対照群（*n* = 25）の疼痛評価（0-10スケール）を比較します。

| グループ | *n* | *Mdn* | IQR |
|----------|-----|-------|-----|
| 治療群 | 25 | 3.00 | 2.00-5.00 |
| 対照群 | 25 | 6.00 | 4.00-7.50 |

APA形式で記述：

> 治療群の疼痛評価（*Mdn* = 3.00, IQR = 2.00-5.00）は対照群（*Mdn* = 6.00, IQR = 4.00-7.50）より低かった。

## ステップ2：Mann-Whitney U結果の報告

### 基本形式

> Mann-Whitney U検定の結果、治療群（*Mdn* = 3.00）の疼痛評価は対照群（*Mdn* = 6.00）より有意に低かった（*U* = 156.50, *z* = -3.24, *p* = .001, *r* = .46）。

### 各要素の説明

**U統計量**：検定の原統計量です。ソフトウェアによって2つのU値のうち小さい方を報告するものと大きい方を報告するものがあります。ソフトウェアのドキュメントを確認し、一貫して使用してください。

**z得点**：Uの標準化値で、大標本での有意性検定に使用されます。差の方向を示す符号（正または負）を含めてください。

**p値**：正確な *p* 値を小数点第3位まで報告します。.001未満の場合は *p* < .001と表記します。小標本では、漸近近似ではなく順列分布からの正確*p*を報告します。

**効果量 *r***：*r* = *z* / &radic;*N* で計算します（*N* は全サンプルサイズ）。Mann-Whitney Uで最も一般的に使用される効果量です。

## 効果量 *r* の理解

Mann-Whitney Uの効果量 *r* は相関係数と同じ解釈基準に従います：

| *r* | 解釈 |
|-----|------|
| .10 | 小さい効果 |
| .30 | 中程度の効果 |
| .50 | 大きい効果 |

例では *r* = .46は中程度〜大きい効果に該当し、2群間の疼痛評価に実質的な差があることを示しています。

### 代替：順位双列相関

一部のジャーナルでは順位双列相関（*r*<sub>rb</sub>）を好みます。この値は-1から+1の範囲を取り、より明確な解釈が可能です：一方のグループから無作為に選んだ観測値がもう一方のグループの観測値を上回る確率を相関スケールに変換したものです。

> *r*<sub>rb</sub> = 1 - (2*U*) / (*n*<sub>1</sub> &times; *n*<sub>2</sub>)

## ステップ3：方向性と実質的有意性の報告

効果の方向を常に明確にしてください：

> 治療群（*Mdn* = 3.00）の疼痛評価は対照群（*Mdn* = 6.00）より有意に低く、治療が患者報告の疼痛軽減に効果的であったことを示している。

## 完全な報告例

> **結果**
>
> 治療群（*n* = 25）と対照群（*n* = 25）の疼痛評価をMann-Whitney U検定で比較した。治療群の疼痛評価（*Mdn* = 3.00, IQR = 2.00-5.00）は対照群（*Mdn* = 6.00, IQR = 4.00-7.50）より有意に低かった（*U* = 156.50, *z* = -3.24, *p* = .001, *r* = .46）。効果量は中程度〜大きい群間差を示した。

## Mann-Whitney U vs. 対応のないt検定：選択基準

Mann-Whitney U検定と対応のないt検定のどちらを選択するかは、2群比較における最も一般的な意思決定の一つです。誤った選択は統計的検定力を低下させたり、誤解を招く結果を生む可能性があります。

### 判断基準

主な考慮事項は、データがt検定の要求する仮定を満たしているかどうかです。対応のないt検定は、従属変数が各グループで近似的に正規分布に従い、両グループが類似した分散を持つこと（分散の等質性）を仮定します。これらの仮定が満たされる場合、t検定が好ましい選択です。真の差を検出するためのより高い統計的検定力を持つためです。

以下の条件のいずれかが当てはまる場合、Mann-Whitney U検定を使用してください：

1. **非正規分布**：Shapiro-Wilk検定で*p* < .05、またはQ-Qプロットで正規性からの顕著な逸脱が観察される。
2. **順序データ**：結果が順序尺度で測定されている（例：リッカート項目、順位データ）。
3. **深刻な外れ値**：極端な値がグループ平均を歪めるが、中央値と順位への影響は最小限。
4. **分布が不明な小標本**：グループあたり*n* < 15-20で正規性を確認できない場合。

### 検定力の比較

すべての仮定が満たされる場合、t検定のほうが強力です。正規分布データに対するMann-Whitney U検定の漸近相対効率（ARE）は約0.955です。これは、Mann-Whitney検定が同じ検定力を達成するために約5%多くの観測値を必要とすることを意味します。しかし、分布が歪んでいる、裾が重い、または外れ値で汚染されている場合、Mann-Whitney U検定はt検定よりもかなり強力になりえます。

### 検定選択のAPA報告

結果セクションで常に検定の選択を正当化してください：

> Shapiro-Wilk検定の結果、治療群（*W* = 0.85, *p* = .002）と対照群（*W* = 0.89, *p* = .012）の両方で正規性からの有意な逸脱が確認された。そのため、対応のないt検定の代わりにMann-Whitney U検定を使用した。

## 信頼区間付きのMann-Whitney U報告

APA第7版は、点推定値とともに信頼区間を報告することを強く推奨しています。Mann-Whitney U検定では、Hodges-Lehmann推定量と効果量の信頼区間を計算できます。

### Hodges-Lehmann推定量

Hodges-Lehmann推定量は、2群間の位置シフトの頑健な推定値を提供します。2群のすべての対ごとの差の中央値として計算されます。中央値の差とは異なり、Hodges-Lehmann推定量はMann-Whitney U統計量と直接関連しており、処理効果の意味のある点推定値を提供します。

例えば、Hodges-Lehmann推定値が-3.00の場合、治療群の値が対照群の値よりも約3単位低いことを示します。

### 位置シフトの信頼区間

Hodges-Lehmann推定量の95%信頼区間は、対ごとの差の順序付き集合に基づいています。R（`wilcox.test`で`conf.int = TRUE`を使用）やSPSSなどのソフトウェアパッケージでこの区間を直接計算できます。

**Hodges-Lehmann CIを含むAPA報告例：**

> Mann-Whitney U検定の結果、治療群（*Mdn* = 3.00）の疼痛評価は対照群（*Mdn* = 6.00）より有意に低かった（*U* = 156.50, *z* = -3.24, *p* = .001）。位置シフトのHodges-Lehmann推定値は-3.00であった（95% CI [-4.50, -1.50]、*r* = .46）。

### 効果量の信頼区間

順位双列相関の信頼区間も報告できます。これにはブートストラップまたは*r*<sub>rb</sub>の分散に関する専門的な公式が必要です。利用可能な場合は報告に含めてください：

> *U* = 156.50, *z* = -3.24, *p* = .001, *r*<sub>rb</sub> = .50, 95% CI [.22, .71]。

信頼区間を含めることで、推定値の精度を伝え、読者が妥当な効果の範囲を評価できるようになり、報告が強化されます。

## Mann-Whitney Uの効果量：完全ガイド

効果量はAPA報告の重要な構成要素です。Mann-Whitney U検定では、順位双列相関と*z*得点から導出した*r*統計量の2つの効果量指標が一般的に使用されます。

### 順位双列相関（*r*<sub>rb</sub>）

順位双列相関は、直接的な確率的解釈が可能なため、Mann-Whitney Uに最も適した効果量です。計算式：

> *r*<sub>rb</sub> = 1 - (2*U*) / (*n*<sub>1</sub> &times; *n*<sub>2</sub>)

この値は-1から+1の範囲を取ります。正の*r*<sub>rb</sub>はグループ1の観測値がグループ2より大きい傾向を示し、その逆も同様です。絶対値は、一方のグループから無作為に選んだ観測値がもう一方を上回る確率を相関指標に変換したものです。

**順位双列相関を含むAPA報告例：**

> Mann-Whitney U検定の結果、リデザインされたインターフェースグループの顧客満足度スコア（*Mdn* = 8.00）は、従来のインターフェースグループ（*Mdn* = 5.50）より有意に高かった（*U* = 89.00, *z* = -3.51, *p* < .001, *r*<sub>rb</sub> = .62）。

### *r* = *z* / &radic;*N* 法

*z*得点から*r*を算出する代替的で広く使用される方法：

> *r* = *z* / &radic;*N*

ここで*N*は両グループを合わせた全サンプルサイズです。この指標はCohen（1988）のベンチマークに従います：

| *r* 値 | 解釈 | 対応するCohen's *d* |
|--------|------|-------------------|
| .10 | 小さい効果 | 0.20 |
| .30 | 中程度の効果 | 0.50 |
| .50 | 大きい効果 | 0.80 |

### 共通言語効果量（CLES）

共通言語効果量（優越確率とも呼ばれる）は、最も直感的な解釈を提供します。一方のグループから無作為に選んだ観測値がもう一方のグループから無作為に選んだ観測値より大きくなる確率を表します。

> CLES = *U* / (*n*<sub>1</sub> &times; *n*<sub>2</sub>)

例えば、CLESが.75の場合、すべての可能なペアの75%でグループ1の観測値がグループ2の観測値を上回ることを意味します。CLESと順位双列相関の関係は：*r*<sub>rb</sub> = 1 - 2 &times; CLES（Uがグループ2がグループ1を上回る回数として定義される場合）。

**CLESを含むAPA報告例：**

> 共通言語効果量は、治療群から無作為に選んだ患者が対照群から無作為に選んだ患者よりも低い疼痛を報告する確率が75%であることを示した（CLES = .75）。

### どちらの効果量を使うべきか

- ソフトウェアが提供する場合や、Mann-Whitney検定に最も解釈しやすい効果量を求める場合は**順位双列相関**（*r*<sub>rb</sub>）を使用してください。
- *z*得点と全サンプルサイズのみがある場合は***r* = *z* / &radic;*N***を使用してください。この方法はより簡単ですが、同順位がある場合に順位双列相関とわずかに異なる場合があります。
- 非専門家に結果を伝える場合や、確率的解釈が研究課題に最も適切な場合は**CLES**を使用してください。

### 信頼区間付きの効果量報告

APA第7版は、可能な場合に効果量の信頼区間の報告を推奨しています：

> 治療群は対照群より有意に高いスコアを示した（*Mdn* = 7.00 vs. *Mdn* = 4.00）、*U* = 112.00, *z* = -2.89, *p* = .004, *r*<sub>rb</sub> = .54, 95% CI [.21, .76]。

## 同順位があるMann-Whitney Uの報告

同順位は、2つ以上の観測値が同じ値を持つ場合に発生します。リッカート尺度や評定尺度などの順序データでは一般的です。同順位は*U*統計量と有意性の計算の両方に影響します。

### 同順位がU統計量に与える影響

同順位がある場合、すべての同順位値にはそれらが占めるべき順位の平均が付与されます。例えば、順位3と4で同順位が発生した場合、両方の値に3.5の順位が付けられます。この中間順位の付与は順位合計を保持しますが、検定統計量の分布を変化させます。

Mann-Whitney Uの標準*z*得点の公式：

> *z* = (*U* - *n*<sub>1</sub>*n*<sub>2</sub>/2) / &radic;(*n*<sub>1</sub> &times; *n*<sub>2</sub> &times; (*n*<sub>1</sub> + *n*<sub>2</sub> + 1) / 12)

同順位がある場合、分母に補正係数が適用されます：

> *z*<sub>補正</sub> = (*U* - *n*<sub>1</sub>*n*<sub>2</sub>/2) / &radic;((*n*<sub>1</sub> &times; *n*<sub>2</sub>) / (*N*(*N* - 1))) &times; ((*N*<sup>3</sup> - *N*) / 12 - &Sigma;(*t*<sub>i</sub><sup>3</sup> - *t*<sub>i</sub>) / 12)

ここで*t*<sub>i</sub>は*i*番目の同順位グループの観測数です。

### 正確p値 vs. 漸近p値

ソフトウェアは通常、2種類の*p*値を提供します：

| 方法 | 使用場面 | 利点 |
|------|---------|------|
| **正確 *p*** | 小標本（グループあたり *n* < 20） | 小標本で正確；分布の仮定不要 |
| **漸近 *p*** | 大標本（グループあたり *n* &ge; 20） | 計算効率が高い；大標本で正確 |

**重要な指針：**
- グループあたり20未満の観測値がある場合、必ず正確*p*値を報告してください。
- 大標本では漸近*p*が許容され、正確値に近い近似値を提供します。
- 同順位が広範な場合（値の15-20%以上）、報告書にその旨を記載し、同順位補正*z*統計量を使用してください。

**同順位補正を含むAPA報告例：**

> リッカート尺度データの順序的性質のため、同順位補正を適用したMann-Whitney U検定を使用した。ハイブリッド勤務グループの仕事満足度（*Mdn* = 4.00, IQR = 3.00-5.00）はオフィス勤務グループ（*Mdn* = 3.00, IQR = 2.00-4.00）より有意に高かった（*U* = 245.00, *z* = -2.67（同順位補正済み）, *p* = .008, *r* = .38）。

## 複数グループのMann-Whitney U：対ごとの比較

Mann-Whitney U検定は正確に2つの独立グループを比較するために設計されていますが、研究者はしばしばノンパラメトリック法で複数のグループを比較する必要があります。この文脈でMann-Whitney Uをいつ、どのように適用するかを理解することが不可欠です。

### 3グループ以上の場合

研究デザインに3つ以上の独立グループが含まれる場合、複数のMann-Whitney U検定を直接実行**しないでください**。このアプローチは第1種の過誤率を膨張させます。例えば、4グループの場合、6つの対ごとの比較が必要となり、ファミリーワイズ過誤率は.05から約.26に上昇します。

代わりに、まず**Kruskal-Wallis H検定**を全体検定（オムニバス検定）として実施してください。Kruskal-Wallis検定が有意な場合にのみ、事後の対ごとの比較に進みます。

### Bonferroni補正付きの対ごとのMann-Whitney U

有意なKruskal-Wallis結果の後、Bonferroni補正付きのMann-Whitney U検定で対ごとの比較を実施できます。有意水準を比較の数で割ります：

> 補正後α = .05 / *k*、ここで*k* = 対ごとの比較数

3グループ（A、B、C）の場合、3つの対ごとの比較（A-B、A-C、B-C）があるため、補正後αは.05 / 3 = .017です。

**APA報告例：**

> Kruskal-Wallis H検定の結果、教授法が試験成績に有意な影響を及ぼすことが示された（*H*(2) = 14.32, *p* < .001, &epsilon;<sup>2</sup> = .16）。Bonferroni補正（α = .017）を適用した事後対ごとのMann-Whitney U検定により、方法A（*Mdn* = 82.00）は方法C（*Mdn* = 68.00）より有意に高い成績を示した（*U* = 45.00, *z* = -3.21, *p* = .001, *r* = .52）。方法AとB（*p* = .089）および方法BとC（*p* = .034）の間には有意差は認められなかった。

### 代替：Dunnの検定

対ごとのMann-Whitney U検定の代替として、Kruskal-Wallis検定の事後手続きとして特別に設計されたDunnの検定があります。Dunnの検定は各ペアごとに再順位付けするのではなく、元のオムニバス順位からの順位和を使用するため、Kruskal-Wallis検定との一貫性を維持します。

## Mann-Whitney U報告のよくある間違い

基本的な報告の誤りに加えて、研究者が頻繁に犯す間違いが統計報告の質を低下させます。

### 間違い1：順位ではなく平均の比較

Mann-Whitney U検定は順位分布を比較するものであり、平均を比較するものではありません。中央値の代わりにグループ平均を報告すると、検定が実際に評価しているものを誤って伝えることになります。

**誤った例：**
> 治療群の平均疼痛スコア（*M* = 3.42, *SD* = 1.86）は対照群（*M* = 5.94, *SD* = 2.13）より低かった（*U* = 156.50, *p* = .001）。

**正しい例：**
> 治療群の疼痛評価（*Mdn* = 3.00, IQR = 2.00-5.00）は対照群（*Mdn* = 6.00, IQR = 4.00-7.50）より低かった（*U* = 156.50, *z* = -3.24, *p* = .001, *r* = .46）。

### 間違い2：分布の仮定の無視

Mann-Whitney U検定は正規性を仮定しませんが、以下を仮定します：

- グループ間の観測値の**独立性**
- 両グループの**類似した分布形状**（結果を中央値の差として解釈する場合）
- 少なくとも**順序水準**の測定

分布形状が大きく異なる場合（例：一方のグループが右に歪み、もう一方が左に歪んでいる）、Mann-Whitney U検定は中央値ではなく*確率的優越性*を比較しています。この場合、中央値ではなく順位和または平均順位を報告してください。

### 間違い3：効果量の欠落

APA第7版のガイドラインに基づき、*U*と*p*のみを報告して効果量を省略することは不完全です。すべての推測的検定には効果量の提示が必要です。

**不完全：** *U* = 156.50, *p* = .001

**完全：** *U* = 156.50, *z* = -3.24, *p* = .001, *r* = .46

### 間違い4：z得点の未報告

*U*統計量だけではサンプルサイズによって値が変わるため、解釈の文脈が不足します。*z*得点は*U*を標準化し、効果量*r*の計算に不可欠です。

### 間違い5：t検定が適切な場合にMann-Whitney Uを使用

一部の研究者は不必要にノンパラメトリック検定をデフォルトで使用します。データが連続型で、近似的に正規分布に従い、等分散であれば、対応のないt検定のほうが統計的検定力が高くなります。パラメトリックの仮定が違反された場合にのみMann-Whitney Uを使用してください。

### 間違い6：検定選択の正当化の未記述

なぜt検定ではなくMann-Whitney U検定を選択したのか、常に説明してください。選択の根拠となった具体的な仮定の違反（例：非正規性、順序データ、外れ値）を記述してください。

**正当化の例：**
> Shapiro-Wilk検定の結果、治療群の疼痛評価は有意に非正規であった（*W* = 0.87, *p* = .005）。そのため、対応のないt検定の代わりにMann-Whitney U検定を使用した。

### 間違い7：正確p値と漸近p値の混同

研究者は、正規近似が不正確な小標本で漸近*p*値を報告することがあります。小標本（グループあたり*n* < 20）では、正確*p*値のほうが信頼性が高いです。どちらのタイプの*p*値を報告しているかを常に明記してください：

**小標本での正しい例：**
> 正確Mann-Whitney U検定の結果、グループ間でスコアに有意差が認められた（*U* = 12.00, exact *p* = .023, *r*<sub>rb</sub> = .52）。

### 間違い8：効果量計算の誤り

誤った公式を使用したり、順位双列相関の符号を誤解したりすることはよくある誤りです。*r*<sub>rb</sub>の符号は*U*の定義方法とどちらのグループをグループ1として扱うかによって変わります。常にグループの中央値と比較して方向を確認してください。

## Mann-Whitney U結果のAPA表形式

複数のグループ比較を報告する場合や詳細な結果を提示する場合、APA形式の表は本文中の記述より効率的です。以下は完全な表の例です。

### 2グループ比較の表形式

**表1**

*治療条件別疼痛評価のMann-Whitney U検定結果*

| 変数 | グループ | *n* | *Mdn* | IQR | *U* | *z* | *p* | *r* |
|------|----------|-----|-------|-----|-----|-----|-----|-----|
| 疼痛評価 | 治療群 | 25 | 3.00 | 2.00-5.00 | 156.50 | -3.24 | .001 | .46 |
| | 対照群 | 25 | 6.00 | 4.00-7.50 | | | | |

### 複数従属変数の表形式

同一の2グループに対して複数の従属変数を検定した場合、1つの表にまとめます：

**表2**

*治療群別患者報告アウトカムのMann-Whitney U検定結果*

| アウトカム | 治療群 *Mdn* (IQR) | 対照群 *Mdn* (IQR) | *U* | *z* | *p* | *r* |
|----------|-------------------|-------------------|-----|-----|-----|-----|
| 疼痛 (0-10) | 3.00 (2.00-5.00) | 6.00 (4.00-7.50) | 156.50 | -3.24 | .001 | .46 |
| 不安 (0-10) | 4.00 (2.50-5.50) | 5.00 (3.50-7.00) | 218.00 | -1.94 | .052 | .27 |
| 可動性 (1-5) | 4.00 (3.00-5.00) | 3.00 (2.00-4.00) | 187.50 | -2.56 | .010 | .36 |
| 睡眠の質 (1-5) | 3.00 (2.00-4.00) | 3.00 (2.00-4.00) | 295.50 | -0.47 | .638 | .07 |

### 表の注記

検定の詳細を明記する表の注記を必ず含めてください：

> *注.* *N* = 50（各グループ25名）。効果量 *r* = *z* / &radic;*N*。有意性は両側検定。

### APA表の作成ガイドライン

- すべての統計記号（*U*, *z*, *p*, *r*, *Mdn*, *n*）を**イタリック体**で表記します。
- **水平線**は表の上部、ヘッダー行の下、表の下部にのみ使用します。
- 垂直線は使用しません。
- *p*値は小数点第3位まで報告します。非常に小さい値は*p* < .001と表記します。
- 有意な*p*値を太字にすることは可能ですが、任意です。

## Mann-Whitney U vs. 他のノンパラメトリック検定：いつ何を使うか

適切なノンパラメトリック検定の選択は研究デザインによって異なります。以下は意思決定ガイドです。

### Mann-Whitney U vs. 対応のないt検定

| 基準 | t検定 | Mann-Whitney U |
|------|-------|----------------|
| 分布 | 正規分布 | 任意の分布 |
| 尺度 | 間隔/比率 | 順序以上 |
| 外れ値 | 影響を受けやすい | 頑健 |
| サンプルサイズ | 任意（正規性充足時） | 任意 |
| 検定力 | 高い（仮定充足時） | やや低い |
| 効果量 | Cohen's *d* | *r* または *r*<sub>rb</sub> |

データが正規性の仮定を満たす場合、t検定のほうが統計的検定力が高くなります。仮定が違反された場合、Mann-Whitney Uが適切な選択です。

### Mann-Whitney U vs. Wilcoxon符号順位検定

| 特徴 | Mann-Whitney U | Wilcoxon符号順位 |
|------|---------------|-----------------|
| デザイン | 独立グループ | 対応/反復測定 |
| 比較 | 異なる2グループ | 同一対象者を2回測定 |
| 例 | 治療群 vs. 対照群 | 事前テスト vs. 事後テスト |
| パラメトリック対応 | 対応のない*t*検定 | 対応のある*t*検定 |

**重要な区別：** Mann-Whitney Uは*被験者間*デザインに、Wilcoxon符号順位は*被験者内*デザインに使用します。

### Mann-Whitney U vs. Kruskal-Wallis H検定

| 特徴 | Mann-Whitney U | Kruskal-Wallis H |
|------|---------------|-----------------|
| グループ数 | 正確に2 | 3以上 |
| 事後検定 | 不要 | Dunnの検定 + Bonferroni補正 |
| パラメトリック対応 | 対応のない*t*検定 | 一元配置分散分析 |
| 効果量 | *r* | イプシロン二乗（*&epsilon;*<sup>2</sup>） |

**重要な区別：** Mann-Whitney Uは2グループ比較に限定されます。3グループ以上の独立グループにはKruskal-Wallis H検定を使用してください。Kruskal-Wallis検定が有意であれば、対ごとのMann-Whitney U検定（Bonferroni補正付き）またはDunnの検定で事後分析を行ってください。

### 意思決定ガイド：適切なノンパラメトリック検定の選択

1. **比較するグループはいくつですか？**
   - 2グループ &rarr; ステップ2へ
   - 3グループ以上 &rarr; Kruskal-Wallis H検定（独立）またはFriedman検定（反復測定）

2. **グループは独立ですか、関連していますか？**
   - 独立（異なる参加者） &rarr; **Mann-Whitney U検定**
   - 関連（同一参加者、事前/事後） &rarr; **Wilcoxon符号順位検定**

3. **ノンパラメトリックを選択する前に確認：**
   - 正規性が違反されていますか？（Shapiro-Wilk *p* < .05）
   - 深刻な外れ値がありますか？
   - データは順序尺度ですか？
   - 非正規分布でグループあたり *n* < 30ですか？

これらのいずれにも該当しない場合、より高い統計的検定力のためにパラメトリック代替法を検討してください。

## よくある質問（FAQ）

### Mann-Whitney U統計量を手計算するにはどうすればよいですか？

両グループのすべての観測値を合わせて小さい値から順位を付けます。各グループの順位を合計します（*R*<sub>1</sub>と*R*<sub>2</sub>）。次に*U*<sub>1</sub> = *n*<sub>1</sub>*n*<sub>2</sub> + *n*<sub>1</sub>(*n*<sub>1</sub> + 1)/2 - *R*<sub>1</sub>と*U*<sub>2</sub> = *n*<sub>1</sub>*n*<sub>2</sub> + *n*<sub>2</sub>(*n*<sub>2</sub> + 1)/2 - *R*<sub>2</sub>を計算します。小さい*U*値を報告します（またはソフトウェアの基準に従う）。

### 順序型リッカート尺度データにMann-Whitney Uを使用できますか？

使用できます。Mann-Whitney Uは生の値ではなく順位に基づくため、リッカート尺度データに適しています。ただし、リッカート尺度では同順位が一般的であるため、同順位補正*z*統計量を使用し、正確p値と漸近p値のどちらを使用したかを報告してください。

### Mann-Whitney U検定の最小サンプルサイズはいくつですか？

技術的にはグループあたり4つの観測値で検定が可能ですが、統計的検定力は非常に低くなります。中程度の効果（*r* = .30）を80%の検定力で検出するには、グループあたり少なくとも20-30の観測値を確保してください。

### 片側検定と両側検定のどちらのp値を報告すべきですか？

データ収集前に強い方向性仮説がある場合を除き、両側*p*値を使用してください。APAガイドラインは両側検定をデフォルトとして推奨しています。片側検定を使用する場合は、その旨を明記し方向性仮説を正当化してください。

### Mann-Whitney UとWilcoxon順位和検定は同じ検定ですか？

はい、2つの独立グループを比較するための数学的に等価な検定です。Mann-Whitney U検定とWilcoxon順位和検定は同一の*p*値を算出します。対応標本に使用するWilcoxon符号順位検定と混同しないでください。

### Mann-Whitney Uの結果が有意でない場合、どのように解釈しますか？

有意でない結果は、2つのグループが異なるという十分な証拠がないことを意味します。効果量を含む完全な結果を報告してください：「グループ間の差は統計的に有意ではなかった（*U* = 298.00, *z* = -0.42, *p* = .674, *r* = .06）。」小さい効果量は、意味のある差がないという解釈を支持します。

### Mann-Whitney Uを2グループ以上に使用できますか？

使用できません。Mann-Whitney Uは正確に2つの独立グループのために設計されています。3グループ以上にはKruskal-Wallis H検定を使用してください。Kruskal-Wallis検定が有意であれば、Bonferroni補正を適用した対ごとのMann-Whitney U検定で事後分析を行ってください。

### 2つのグループのサンプルサイズが大きく異なる場合はどうすればよいですか？

Mann-Whitney Uは不均等なサンプルサイズを処理できますが、極端な不均衡（例：10 vs. 100）は検定力を低下させ、漸近*p*値の精度に影響する可能性があります。非常に不均等なサイズの場合は、正確*p*値を使用し、*r* = *z* / &radic;*N*よりも順位双列相関がより安定した効果量指標であるかどうかを検討してください。

### Hodges-Lehmann推定量とは何ですか？いつ報告すべきですか？

Hodges-Lehmann推定量は、2群間のすべての対ごとの差の中央値です。Mann-Whitney U検定と直接結びついた位置シフトの頑健な測度を提供します。グループ差の大きさを元の測定単位で定量化したい場合、特にAPA第7版が効果の信頼区間を推奨する文脈において報告してください。

### Mann-Whitney Uはデータの同順位をどのように処理しますか？

複数の観測値が同じ値を持つ場合、それらが占めるべき順位の平均（中間順位）が付与されます。z得点の計算時にU統計量の分散に同順位補正係数が適用されます。ほとんどの統計ソフトウェアがこの補正を自動的に適用します。同順位が広範な場合（観測値の15-20%以上）、報告書に同順位補正について記載してください。

### Mann-Whitney U検定で平均と中央値の両方を報告できますか？

記述的な完全性のために中央値とともに平均を報告することは可能ですが、Mann-Whitney U検定の主要な記述統計量は常に中央値とIQRであるべきです。平均を報告する場合は、Mann-Whitney U検定は平均差を検定するものではなく、平均は追加的な文脈のためにのみ提供されていることを明確にしてください。

### 片側Mann-Whitney U検定と両側Mann-Whitney U検定の違いは何ですか？

両側検定は2つのグループの分布がいずれの方向にも異なるかを評価し、片側検定は一方の特定のグループがより大きな値を生み出す傾向があるかを評価します。ほとんどの研究では両側検定がデフォルトです。強い事前の方向性仮説がある場合にのみ片側検定を使用し、方法セクションでこの選択を明示的に報告してください。
