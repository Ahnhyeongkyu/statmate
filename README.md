# StatMate - Free APA Statistics Calculator

**[statmate.org](https://statmate.org)**

StatMate is a free, trilingual (English / Korean / Japanese) online statistics calculator that formats results in APA 7th edition style. Built for researchers, students, and academics who need accurate statistical analysis with publication-ready output.

## Features

- **20 Statistical Calculators** - t-test, ANOVA, chi-square, correlation, regression, factor analysis, and more
- **APA 7th Edition Output** - Copy-paste ready results formatted to APA standards
- **Trilingual** - Full support for English, Korean, and Japanese
- **Free PDF Export** - Download results as formatted PDF
- **Data Visualization** - Built-in charts (box plots, scatter plots, residual plots, etc.)
- **CSV Upload** - Import data from CSV/TSV/TXT files
- **Assumption Checks** - Automatic normality and homogeneity tests
- **Post-hoc Analysis** - Bonferroni, Dunn's, Nemenyi corrections
- **URL Sharing** - Share results via URL parameters
- **Dark Mode** - Full dark mode support
- **PWA** - Install as a desktop/mobile app

## Calculators

| Category | Tests |
|----------|-------|
| Parametric | Independent/Paired t-Test, One-Sample t-Test, One-Way ANOVA, Two-Way ANOVA, Repeated Measures ANOVA |
| Non-parametric | Mann-Whitney U, Wilcoxon Signed-Rank, Kruskal-Wallis H, Friedman |
| Categorical | Chi-Square, Fisher's Exact, McNemar |
| Regression | Simple Regression, Multiple Regression, Logistic Regression |
| Correlation | Pearson/Spearman Correlation |
| Other | Descriptive Statistics, Sample Size/Power, Cronbach's Alpha, Factor Analysis (EFA) |

## Tech Stack

- [Next.js](https://nextjs.org) 16 with Turbopack
- TypeScript
- Tailwind CSS v4
- [next-intl](https://next-intl-docs.vercel.app/) for i18n
- Deployed on [Vercel](https://vercel.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
